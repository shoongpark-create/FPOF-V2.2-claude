# 회의록 정리 (Meeting Notes)

회의 트랜스크립트를 구조화된 회의록으로 변환합니다. 결정 사항, 액션 아이템, 후속 조치를 포함합니다.

## 사용법
`/meeting <트랜스크립트 또는 회의 노트>` — 텍스트, 파일, 오디오 요약 등 모든 형식을 지원합니다.

## 절차

### Step 1: 트랜스크립트 수집
1. `.fpof-state.json`에서 현재 시즌과 단계 확인
2. 입력 수용:
   - 전체 트랜스크립트 (Otter, Clova, Zoom 등)
   - 수기 메모
   - 오디오 요약
   - 복수 입력 (트랜스크립트 + 개인 메모)

### Step 2: 구조화된 추출
`skills/pm-execution/summarize-meeting.md` 참조:
- **참석자**: 이름/역할 식별
- **논의 주제**: 주요 안건별 정리
- **결정 사항**: 명확한 합의나 결론
- **액션 아이템**: 담당자와 데드라인 포함
- **미해결 질문**: 후속 필요 항목
- **핵심 인용구**: 보존할 가치가 있는 발언

### Step 3: 회의록 생성
- 3~5문장 요약 (회의에 참석하지 않은 사람도 30초 안에 파악)
- 결정 사항 리스트 (맥락과 근거 포함)
- 액션 아이템 테이블 (담당자, 데드라인, 상태)
- 주제별 논의 하이라이트
- 미해결 질문 및 다음 단계

### Step 4: 산출물 저장
산출물을 `output/[시즌]/_season/do_meeting-notes.md`에 저장한다.
`.fpof-state.json`의 artifacts 배열에 파일 경로 추가.

### Step 5: 다음 단계 제안
- "결정 사항을 기반으로 **이해관계자 업데이트**를 작성하시겠어요?"
- "액션 아이템으로 **티켓을 생성**하시겠어요?"
- "참석자에게 보낼 **이메일 요약**을 작성하시겠어요?"
