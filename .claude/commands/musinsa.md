# 무신사 랭킹 수집

무신사 랭킹 API에서 카테고리별/기간별/성별/연령별 상품 랭킹 데이터를 수집하고 엑셀로 저장합니다.

## 사용법
`/musinsa [옵션]` — 옵션을 생략하면 카테고리/기간/성별/연령을 물어봅니다.

## 절차

1. `system/skills/data/musinsa-ranking.md` 스킬 파일을 읽어 실행 절차 확인
2. **AskUserQuestion으로 필터 선택** (카테고리, 기간, 성별, 연령대)
3. 사용자 선택을 파라미터로 변환하여 `system/scripts/musinsa-crawler/crawler.py` 실행
4. 수집 결과 보고 (상위 상품 요약 + 와키윌리 관점 인사이트)

## 필터 옵션
- **카테고리**: 전체(000), 상의(001), 아우터(002), 바지(003), 가방(004), 신발(103), 소품(101), 원피스/스커트(100), 뷰티(104), 속옷/홈웨어(026), 스포츠/키즈(017)
- **기간**: 실시간(REALTIME), 일간(DAILY), 주간(WEEKLY), 월간(MONTHLY)
- **성별**: 전체(A), 남성(M), 여성(F)
- **연령대**: 전체, 19세이하, 20-24, 25-29, 30-34, 35-39, 40+

## 예시
```
/musinsa                          → 필터 선택 → 수집
/musinsa 상의 주간 여성             → 상의/주간/여성 수집
/musinsa 랭킹 가져와줘              → 필터 선택 → 수집
```

## 출력
- 엑셀(.xlsx): 순위, 브랜드, 상품명, 상품이미지(셀 삽입), 가격, 할인율, 뱃지, 실시간 정보, 상품URL, 이미지URL
- 저장 위치: `workspace/musinsa-ranking/ranking_YYYYMMDD/`
