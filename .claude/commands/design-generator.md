# AI 디자인 제너레이터

텍스트/스케치/레퍼런스를 입력받아 와키윌리 브랜드 DNA에 맞는 디자인 시안 세트(플랫 스케치, 착장 룩, 그래픽, 컬러웨이)를 생성합니다.

## 사용법
`/design-generator [옵션]` — 옵션 생략 시 아이템/라인/입력소스/배리에이션 수를 물어봅니다.

## 절차

1. `system/skills/creative/design-generator.md` 스킬 파일을 읽어 실행 절차 확인
2. **AskUserQuestion으로 디자인 브리프** (아이템, 라인, 입력 소스, 배리에이션 수)
3. 브리프 파싱 & 브랜드 DNA 필터
4. 디자인 컨셉 배리에이션 생성
5. 플랫 스케치 프롬프트 (정면/후면/디테일)
6. 착장 룩 프롬프트 (스트릿/스튜디오/에디토리얼/키치)
7. 그래픽/프린트 프롬프트 (IP 캐릭터 연동)
8. 컬러웨이 매트릭스

## 입력 소스
- **자연어 브리프**: "키치 오버사이즈 후드" 같은 텍스트
- **아이템 브리프 파일**: 기존 design_item-brief.md
- **스케치 이미지**: 손 스케치 사진 → 디자인 변환
- **레퍼런스 이미지**: 참고 이미지 → 와키윌리 재해석

## 예시
```
/design-generator                           → 브리프 설정 → 시안 생성
/design-generator 키키 그래픽 티 5개         → KIKY 그래픽 티 5 배리에이션
/design-generator 스케치 후디                → 스케치 이미지 → 디자인 시안
```

## 출력
- 디자인 제너레이터 시트: `workspace/[시즌]/[프로젝트]/design_generator-sheet.md`
- 배리에이션 테이블, 프롬프트 세트, 컬러웨이 매트릭스, design-spec 연동 데이터
