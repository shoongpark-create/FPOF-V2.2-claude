# 핀터레스트 이미지 수집

Pinterest에서 키워드 기반으로 이미지를 수집하고 키워드별 폴더로 자동 분류합니다.

## 사용법
`/pinterest [키워드 또는 이미지]` — 키워드를 생략하면 무엇을 수집할지 물어봅니다.

## 절차

1. `system/skills/creative/pinterest-crawl.md` 스킬 파일을 읽어 실행 절차 확인
2. 사용자 입력(텍스트/이미지)을 분석하여 검색 키워드 설계
3. **5요소 분석 결과와 키워드를 사용자에게 확인** (확인 후 진행)
4. `system/scripts/pinterest-crawler/crawler.py`를 `--headless` 모드로 실행
5. 수집된 이미지를 키워드별 폴더로 자동 분류
6. 결과 보고

## 키워드 설계 원칙
- 텍스트: `[소재/디테일] + [아이템] + [맥락/스타일]` 3~5단어 조합
- 이미지: ①소재 ②형태 ③디테일 ④컬러 ⑤맥락 순서로 추출
- 영문 메인 + 한국어 보조 병렬 검색
- 수량 미지정 시 기본 30장

## 예시
```
/pinterest dopamine dressing        → 키워드 확인 → 30장 수집
/pinterest 텐션업코디 50            → 키워드 확인 → 50장 수집
/pinterest [이미지 첨부]            → 5요소 분석 → 키워드 확인 → 수집
```

## 추가 기능
- 수집 후 "필터링해줘"라고 하면 Claude Vision AI 브랜드 적합도 필터 실행
- "전체 카테고리 크롤링해줘"라고 하면 7개 브랜드 카테고리 일괄 수집
