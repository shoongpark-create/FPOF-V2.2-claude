# 경쟁 배틀카드 (Competitive Battlecard)

특정 경쟁사 대비 세일즈/마케팅용 배틀카드를 작성합니다.

## 사용법
`/battlecard <와키윌리 vs 경쟁사명>` — 경쟁사를 지정하여 1:1 비교 배틀카드를 생성합니다.

## 절차

### Step 1: 매치업 파악
1. `.fpof-state.json`에서 현재 시즌과 단계 확인
2. `presets/wacky-willy/brand.config.json` 참조 — 포지셔닝, 차별점
3. `presets/wacky-willy/channels.json` 참조 — 채널별 경쟁 상황
4. 사용자에게 확인:
   - 비교 대상 경쟁사
   - 고객 유형 (어떤 고객이 두 브랜드를 비교하는지)
   - 승/패 데이터 또는 세일즈 피드백 유무

### Step 2: 경쟁사 리서치 및 배틀카드 작성
`skills/pm-gtm/competitive-battlecard.md` 참조:

배틀카드 구성:
- **퀵 서머리**: 우리가 이기는 상황 / 지는 상황 / 핵심 차별점
- **포지셔닝**: 그들의 메시지 vs 우리의 카운터 포지셔닝
- **기능/상품 비교표**: 카테고리별 비교 (승리 / 패배 / 동등)
- **가격 비교**: 가격대, 가성비, 프로모션 전략 비교
- **이의 처리**: 주요 이의와 대응 화법 + 증거
- **심을 지뢰**: 고객에게 물어볼 질문 (경쟁사 약점 노출)
- **예상 질문**: 경쟁사가 유도할 질문 + 대응법
- **승/패 패턴**: 이기는 이유 Top 3, 지는 이유 Top 3
- **대화 시작점**: 경쟁사 사용 중인 고객 / 동시 평가 고객별 접근법

### Step 3: 산출물 저장
산출물을 `output/[시즌]/_season/do_battlecard-[경쟁사명].md`에 저장한다.
`.fpof-state.json`의 artifacts 배열에 파일 경로 추가.

### Step 4: 다음 단계 제안
- "다른 경쟁사에 대한 **배틀카드도 추가로** 만드시겠어요?"
- "전체 시장을 대상으로 **경쟁 환경 분석**을 실행하시겠어요?"
- "배틀카드 기반으로 **고객 대면 비교 콘텐츠**를 만드시겠어요?"
