# 비주얼 팩토리

상품 이미지를 입력받아 채널별 비주얼 에셋 풀세트(프롬프트, 코디, 카피)를 자동 설계합니다.

## 사용법
`/visual-factory [옵션]` — 옵션을 생략하면 아이템/채널/스타일/카피 연동을 물어봅니다.

## 절차

1. `system/skills/marketing/visual-factory.md` 스킬 파일을 읽어 실행 절차 확인
2. **AskUserQuestion으로 에셋 주문서** (아이템, 채널, 스타일, 카피 연동)
3. 채널 규격 로드 (무신사/자사몰/인스타/룩북)
4. 비주얼 디렉션 설계 (visual-identity.json 기반)
5. 에셋 매트릭스 생성 (아이템 × 채널 × 컷)
6. 이미지 생성 프롬프트 설계
7. 코디 조합 설계 (BTA 크로스)
8. 카피 연동 (PDP + SNS)

## 채널 규격
- **무신사**: 860×1148px, 화이트 배경, 5~12컷
- **자사몰**: 히어로 1920×800 + PDP 1000×1300, 3~10컷
- **인스타**: 피드 1080² + 스토리 1080×1920, 2~8컷
- **룩북**: 에디토리얼 2000×3000, 3~10컷

## 비주얼 스타일
- **스트릿 무드**: 도심 배경, 자연광, 움직임 포즈
- **스튜디오 클린**: 화이트 배경, 제품 중심
- **키치 맥시멀**: 비비드 배경, 그래피티, IP 캐릭터
- **시즌 테마 맞춤**: 현재 시즌 컨셉 기반

## 예시
```
/visual-factory                            → 주문서 설정 → 전체 실행
/visual-factory 그래픽 티셔츠 무신사          → 그래픽 티 무신사용 에셋 생성
/visual-factory 전체 컬렉션 인스타            → 컬렉션 인스타 비주얼 세트
```

## 출력
- 비주얼 팩토리 시트: `workspace/[시즌]/[프로젝트]/do_visual-factory-sheet.md`
- 에셋 매트릭스, 프롬프트 세트, 코디 조합, PDP/SNS 카피
