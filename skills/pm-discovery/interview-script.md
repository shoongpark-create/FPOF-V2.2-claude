# 고객 인터뷰 스크립트 (Customer Interview Script)

> 코어 타겟(18~25세) 고객 인터뷰를 위한 구조화된 스크립트 -- The Mom Test 방법론 기반, JTBD 탐색 질문, 웜업-핵심 탐색-마무리 섹션 포함

## 언제 사용
- 코어 타겟(18~25세 트렌드리더) 고객 인터뷰를 준비할 때
- 시즌 기획을 위한 디스커버리 리서치를 계획할 때
- 인터뷰 가이드를 체계적으로 작성할 때
- 고객 니즈/페인 포인트를 정성적으로 탐색할 때

## FPOF 컨텍스트
- 이 스킬은 와키윌리의 코어 타겟 고객 인터뷰 준비에 활용합니다
- Continuous Discovery의 Stage 1 (Explore)에서 고객 직접 접점을 통한 인사이트 수집 도구
- 관련 프리셋: brand.config.json (전략), personas.json (코어 타겟), categories.json (카테고리)

## 사전 준비
- `.fpof-state.json`에서 현재 시즌 및 PDCA 단계 확인
- `personas.json`에서 코어 타겟 프로필, 니즈, 페인 포인트 확인
- `brand.config.json`에서 시즌 목표/KPI 확인
- 페르소나, 가설 목록, 상품 브리프, 이전 인터뷰 노트가 있으면 먼저 읽기

## 도메인 컨텍스트

고객 인터뷰는 **Continuous Discovery Stage 1 (Explore)**의 핵심 소스 중 하나입니다. 다른 소스: 이해관계자 인터뷰, 사용 분석, 데이터 분석, 서베이, 시장 트렌드, SEO/SEM 분석. PM은 사용자, 이해관계자, 엔지니어, 디자이너에 직접 접근해야 합니다 -- "프록시 없이." **Product Trio** (PM + 디자이너 + 엔지니어 -- Teresa Torres)가 디스커버리에 함께 참여해야 합니다.

**The Mom Test 규칙** (Rob Fitzpatrick):
- **그들의 삶**에 대해 물어라, 당신의 아이디어에 대해서가 아니라
- **과거**에 대해 물어라, 미래가 아닌 ("X를 사용하시겠어요?"는 쓸모없다)
- **덜 말하고, 더 들어라** -- 80/20 비율 목표
- 인터뷰 중 **절대 피칭하지 마라**
- **강한 감정**을 찾아라 -- 진짜 고통이나 기쁨의 신호
- **칭찬은 노이즈** -- "멋져 보여요!"는 아무것도 알려주지 않는다

## 실행 절차

### Step 1: 리서치 목표 명확화
- 팀이 답을 얻어야 하는 구체적 질문은?
- 이 리서치가 어떤 의사결정에 활용되는가?
- 어떤 가정이 검증 필요한가?

### Step 2: 인터뷰 스크립트 작성

#### 오프닝 (2~3분)
- 자기소개와 목적 (학습, 판매 아님)
- 기대 설정: "맞고 틀린 답은 없습니다. 경험에서 배우려 왔습니다."
- 녹음 허가 요청 (해당 시)
- 가용 시간 확인

#### 웜업: 맥락 & 배경 (5분)
- "평소 패션/옷에 대해 어떤 관심을 가지고 계세요?"
- "온라인에서 옷을 얼마나 자주, 어떤 채널에서 구매하세요?"
- 목표: 라포 형성 및 맥락 이해

#### 핵심 탐색: Jobs to Be Done (15~20분)

**현재 상황과 행동** (과거 시제, 구체적 사례):
- "[우리가 탐색하는 것]을 마지막으로 했을 때를 떠올려 주세요. 어떤 일이 있었나요?"
- "어떤 도구나 방법을 사용했나요?"
- "얼마나 걸렸나요? 누구와 함께했나요?"

**페인 포인트와 불만** (관찰, 유도 아닌):
- "그 과정에서 가장 어려웠던 부분은?"
- "마법의 지팡이가 있다면 무엇을 바꾸고 싶으세요?"
- "이를 해결하려고 어떤 시도를 했나요? 결과는?"

**원하는 결과** (그들의 언어로):
- "이 영역에서 '좋은 것'은 어떤 모습인가요?"
- "잘 되고 있다면 어떻게 알 수 있나요?"

**지불 의사 / 우선순위** (Skin in the Game):
- "현재 이것에 얼마나 시간/돈을 쓰고 계세요?"
- "더 나은 솔루션을 찾아본 적 있나요? 무엇을 찾으셨나요?"
- "이것이 해결된다면 무엇을 포기할 수 있나요?"

#### 프로빙 기법
흥미로운 실마리를 발견했을 때 사용:
- **"그것에 대해 더 말씀해 주세요"** -- 어떤 주제든 열기
- **"왜요?"** (부드럽게, 2~3회) -- 근본 원인 도달
- **"구체적인 예를 들어주실 수 있나요?"** -- 의견에서 사실로
- **"그 다음엔 어떻게 됐나요?"** -- 스토리 따라가기
- **"그때 기분이 어땠나요?"** -- 감정적 강도 포착

#### 마무리 (3~5분)
- "제가 미처 물어보지 못한 중요한 것이 있나요?"
- "이 주제에 대해 제가 이야기해볼 만한 분이 또 있을까요?"
- 시간 감사
- 다음 단계 공유 (해당 시)

### Step 3: 스크립트 커스터마이징
특정 상품 영역, 페르소나, 리서치 목표에 맞게 질문을 조정. 가용 인터뷰 시간에 따라 섹션 추가/제거.

### Step 4: 노트 테이킹 템플릿 포함
```
참여자: [이름 / ID]
날짜: [날짜]
핵심 Job: [달성하려는 것]
현재 솔루션: [현재 사용하는 것]
가장 큰 고통: [#1 불만]
원하는 결과: [성공의 모습]
지불 의사: [투자하는/투자할 금액]
서프라이즈 발견: [예상치 못한 것]
후속 조치: [다음 단계]
```

### Step 5: 와키윌리 적용 필터링
- 코어 타겟(18~25세 트렌드리더)에 맞는 인터뷰 톤과 언어
- 키치 스트릿 / IP 유니버스 관련 탐색 질문 포함
- 무신사, 인스타그램 등 코어 타겟 접점 채널 관련 질문
- 브랜드 인지/선호도 관련 자연스러운 탐색 (피칭 없이)
