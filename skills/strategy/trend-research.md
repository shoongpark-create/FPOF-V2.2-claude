# 트렌드 리서치

> 시장 리서처가 글로벌/로컬 트렌드를 수집하고 와키윌리에 적용 가능한 방향을 제시하는 전문 매뉴얼

## 언제 사용
- "요즘 트렌드 뭐야?", "경쟁사 뭐 하고 있어?", "TikTok에서 뜨는 거 뭐야?"
- PDCA Plan 단계 첫 번째 작업

## 사전 준비
1. `.fpof-state.json` → 현재 시즌 확인
2. `presets/wacky-willy/brand.config.json` → 브랜드 전략 방향 (트렌드 적합성 판단 기준)
3. `presets/wacky-willy/personas.json` → 코어 타겟 18~25세의 관심사/미디어 소비
4. `knowledge/` → 이전 시즌 트렌드 적중률 (있을 경우)

## 실행 절차

### Step 1: Macro Trend 수집
글로벌 패션 위크, 런웨이에서 반복 등장하는 테마를 정리한다.
- 파리/밀라노/뉴욕/서울 컬렉션의 공통 키워드
- 소재/실루엣/컬러의 시즌 흐름
- 라이프스타일 트렌드 (음악, 아트, 서브컬처)
- 분석 기준: 와키윌리의 Kitsch Street & IP Universe 컨셉과의 접점

### Step 2: Micro Trend 수집
SNS/TikTok에서 급부상하는 스타일, 밈, 해시태그를 추적한다.
- TikTok 바이럴 스타일 (조회수 100만+ 기준)
- 인스타그램 해시태그 성장률
- Pinterest 검색 트렌드
- 분석 기준: 지속성 예측 (일시적 밈 vs 성장 중인 트렌드 vs 안정기 트렌드)

### Step 3: 경쟁사 동향 분석
주요 경쟁 브랜드의 시즌 전략을 파악한다.
- 직접 경쟁: 같은 가격대/타겟의 스트릿/캐릭터 브랜드
- 간접 경쟁: 코어 타겟이 관심 갖는 상위/하위 포지션 브랜드
- 분석 항목: 시즌 테마, 신상품 방향, 마케팅 캠페인, 가격 변동

#### 5대 경쟁사 프로파일링 (PM-Skills 통합)
직접 경쟁 브랜드 5개를 선정하여 아래 프레임으로 심층 분석한다.

**경쟁사별 프로파일:**
- 브랜드명, 설립 연도, 규모/매출 추정
- 주요 타겟 세그먼트 및 포지셔닝 전략
- 추정 시장 점유율 또는 고객 베이스 규모
- GTM 전략 (유통 채널, 마케팅 접근법)

**강점 분석:**
- 핵심 역량 및 고유 경쟁 우위
- 고객 가치 제안 (VP)
- 고객 충성도 및 리텐션 시그널

**약점 및 갭 분석:**
- 미충족 고객 니즈, 누락된 카테고리
- 알려진 한계점 또는 고객 불만 영역
- 포지셔닝 갭 (와키윌리가 공략 가능한 영역)

**차별화 기회 도출:**
- 경쟁사 전체에 걸친 미충족 고객 니즈
- 타겟 세그먼트 중 경쟁사가 소홀히 하는 영역
- 와키윌리의 IP Universe / Kitsch Street 컨셉으로 선점 가능한 포지셔닝 공간
- 채널/마케팅 접근법 중 경쟁사가 미활용하는 방식

### Step 4: 소비자 시그널 분석
검색 트렌드, 커뮤니티 반응, 리뷰 키워드에서 실수요를 읽는다.
- 네이버/구글 검색 트렌드 (패션 관련 키워드)
- 커뮤니티 반응 (무신사/에이블리/인스타 댓글)
- 리뷰 키워드 분석 (긍정/부정 패턴)

### Step 5: 와키윌리 적용 필터링
수집한 트렌드를 와키윌리 필터로 걸러낸다.
- 브랜드 DNA(Kitsch Street & IP Universe)와 충돌하지 않는가?
- 코어 타겟(18~25세 트렌드리더)이 반응할 트렌드인가?
- IP 캐릭터와 결합 가능한가?
- 기존 카테고리(유니섹스/우먼스/용품)에서 구현 가능한가?

### Step 6: 시장 규모 추정 (PM-Skills 통합)
와키윌리가 속한 시장의 규모를 TAM/SAM/SOM 프레임워크로 추정한다.

**TAM (Total Addressable Market) — 전체 시장:**
- Top-down 추정: 국내 스트릿/캐릭터 패션 전체 시장 규모 (산업 보고서 기반)
- Bottom-up 추정: 타겟 고객 수 × 평균 구매 단가 × 구매 빈도
- 두 방식의 교차 검증 및 조정

**SAM (Serviceable Addressable Market) — 접근 가능 시장:**
- TAM 중 와키윌리가 실제 서비스 가능한 범위
- 제약 조건: 지역, 채널(channels.json 기준 6개 채널), 가격대, 카테고리
- SAM = TAM의 몇 % (근거 포함)

**SOM (Serviceable Obtainable Market) — 획득 가능 시장:**
- 1~3년 내 현실적으로 확보 가능한 점유율
- 근거: 현재 경쟁 포지션, GTM 역량, 경영목표(brand.config.json)
- SOM = SAM의 몇 % (근거 포함)

**시장 규모 요약 테이블:**
| 지표 | 현재 추정 | 2~3년 후 전망 |
|------|----------|-------------|
| TAM  |          |             |
| SAM  |          |             |
| SOM  |          |             |

**성장 동인 & 핵심 가정:**
- 시장 확대/축소 가능 요인 (트렌드, 인구통계, K-컬처 성장 등)
- 핵심 가정과 신뢰도 (높음/중간/낮음)
- 추정치를 검증하기 위한 데이터 소스 추천
