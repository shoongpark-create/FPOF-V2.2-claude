# 코호트 분석

> 고객 코호트별 재구매율, 리텐션, 피처 채택 패턴을 분석합니다 — pm-skills 원본 프레임워크를 FPOF 패션 하우스 컨텍스트에 맞게 커스터마이징

## 언제 사용
- 시즌별/월별 신규 고객 코호트의 재구매 패턴을 분석할 때
- 채널별 고객 리텐션 커브를 비교하고 이탈 원인을 파악할 때
- IP 콜라보/캠페인 유입 고객의 장기 가치를 평가할 때
- PDCA Check 단계에서 고객 충성도 및 LTV를 검증할 때

## FPOF 컨텍스트
- 이 스킬은 와키윌리 브랜드의 고객 코호트별 행동 패턴 분석에 활용합니다
- 관련 프리셋: `channels.json`(6개 유통채널), `personas.json`(UNI/WOMAN 페르소나, 고객 인사이트)
- 패션 브랜드 특성상 시즌 코호트(SS/FW 첫 구매 시점), 채널 코호트(자사몰/무신사/29CM), 유입 경로 코호트(캠페인/오가닉)가 핵심 분석 단위입니다
- 5대 경영목표 중 "글로벌 대응 강화 — 해외 재구매 비중"과 직접 연결됩니다

## 사전 준비
1. `.fpof-state.json` → 현재 시즌 확인
2. `channels.json` → 채널별 매출 구조 및 고객 분포 참조
3. `personas.json` → UNI/WOMAN 페르소나 특성 참조
4. 코호트 데이터(CSV, Excel, JSON) 또는 분석 대상 설명 준비

## 실행 절차

### Step 1: 데이터 읽기 및 검증
코호트 데이터를 수집하고 품질을 점검합니다.
- CSV, Excel, JSON 데이터 파일 수집 및 구조 확인
- 코호트 식별자, 기간, 참여 지표 확인
- 결측값 및 데이터 품질 이슈 점검
- 핵심 통계 요약 (코호트 크기, 날짜 범위, 사용 가능한 지표)

### Step 2: 정량 분석 실행
코호트별 핵심 지표를 산출합니다.
- 코호트별 리텐션율 및 재구매율 계산
- 리텐션 커브, 이탈 패턴, 이상치 식별
- 채널별/페르소나별 구매 빈도 분석
- 월별(MoM) 또는 기간별 변화율 계산
- 필요 시 pandas/numpy 기반 Python 분석 스크립트 생성

### Step 3: 시각화 생성
분석 결과를 시각적으로 표현합니다.
- 리텐션 히트맵 (코호트 x 기간)
- 코호트별 리텐션 커브 라인 차트
- 채널별 재구매율 비교 차트
- 이탈 포인트 및 참여 트렌드 시각화

### Step 4: 인사이트 및 패턴 식별
의미 있는 패턴을 발견합니다.
- 특정 코호트의 초기 이탈 패턴
- 후기 참여 변화 (장기 충성 고객 특성)
- 캠페인/IP 콜라보 유입 코호트의 차별적 행동
- 시즌성/시간적 트렌드
- 놀라운 발견 및 예상 외 패턴 강조

### Step 5: 후속 리서치 제안
추가 분석 및 액션을 권고합니다.
- 이탈 고객 대상 인터뷰/서베이 설계
- 충성 코호트의 행동 특성 심층 분석
- 세션 리플레이를 통한 핵심 인터랙션 패턴 관찰
- 고 vs 저 리텐션 코호트의 Win/Loss 분석
- A/B 테스트 또는 기능 실험 제안

### Step 6: 와키윌리 적용 필터링
분석 결과를 와키윌리 브랜드 렌즈로 필터링합니다.
- UNI/WOMAN 페르소나별 코호트 차이 해석
- 채널별 리텐션 패턴 비교 (자사몰 vs 플랫폼)
- IP 캐릭터 콜라보 유입 고객의 LTV 평가
- 글로벌 고객 코호트의 재구매 행동 분석
- 5대 경영목표와의 연결점 도출
