# NDA 작성

> 인플루언서/콜라보/공급업체 대상 비밀유지계약서(NDA)를 작성합니다 — pm-skills 원본 프레임워크를 FPOF 패션 하우스 컨텍스트에 맞게 커스터마이징

## 언제 사용
- 인플루언서 시딩/콜라보 전 NDA가 필요할 때
- 신규 원단/제조 공급업체와 기밀 정보 공유 전
- IP 캐릭터 라이선싱/콜라보 파트너와 계약할 때
- 시즌 기획 정보를 외부 파트너에게 사전 공유할 때

## FPOF 컨텍스트
- 이 스킬은 와키윌리 브랜드의 인플루언서, 콜라보 파트너, 공급업체 대상 NDA 작성에 활용합니다
- 패션 브랜드 특성상 시즌 디자인, 컬렉션 기획, IP 캐릭터, 가격 정책 등이 주요 기밀 대상입니다
- 글로벌 확장 시 해외 파트너와의 NDA는 관할권(한국법/외국법) 선택이 중요합니다

## 중요 면책 조항
**이 산출물은 정보 제공 목적이며 법률 자문을 구성하지 않습니다. 최종 문서는 반드시 자격을 갖춘 변호사의 검토를 받으세요. NDA는 법적 구속력이 있는 계약이므로 전문 법률 검토가 필수입니다.**

## 사전 준비
1. `.fpof-state.json` → 현재 시즌 확인
2. 계약 당사자 정보 (회사명, 주소, 대표자)
3. 공유할 기밀 정보 유형 명확화
4. 관할 법원/준거법 결정

## 실행 절차

### Step 1: 요구사항 확인
NDA 작성 전 핵심 사항을 파악합니다.
- 양 당사자 모두 법인인가, 개인인가?
- 어떤 유형의 정보가 공유되는가? (시즌 디자인, 가격, IP 등)
- 단방향(한쪽만 공유) NDA인가, 양방향(상호) NDA인가?
- 관할 법원은? (서울중앙지방법원 / 해외 법원)
- NDA 유효기간은?

### Step 2: NDA 구조 설정
표준 섹션으로 구성합니다.

1. **전문** (당사자, 정의, 체결일)
2. **정의** ("기밀정보"란?)
3. **비밀유지 의무** (핵심 의무)
4. **허용된 공개** (비밀유지 예외)
5. **유효기간 및 존속기간**
6. **자료 반환 또는 폐기**
7. **구제수단** (위반 시 결과)
8. **일반조항** (준거법, 관할, 분리가능성)

### Step 3: 와키윌리 기밀 정보 유형 특정
패션 브랜드 고유의 기밀 정보를 명시합니다.
- 시즌 컬렉션 디자인 및 기획 자료
- 프린트/패턴/그래픽 디자인 원본
- IP 캐릭터(키키 등) 미공개 자료
- 가격 정책 및 원가 구조
- 유통 전략 및 채널별 조건
- 마케팅 캠페인 사전 기획
- 고객 데이터 및 분석 결과

### Step 4: 평이한 언어로 작성
명확하고 접근하기 쉬운 언어로 작성합니다.
- 법률 전문 용어보다 명확한 표현 우선
- 용어 최초 사용 시 정의 포함
- [법률 검토 필요] 마크로 전문 검토가 필요한 부분 표시

### Step 5: 맥락 설명 제공
각 섹션의 의미와 고려사항을 설명합니다.
- 각 섹션이 왜 중요한지
- 당사자가 결정해야 할 사항
- 일반적인 함정이나 고려사항
