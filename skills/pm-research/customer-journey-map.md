# 고객 여정 맵

> 와키윌리 6개 채널별 고객 여정을 매핑하고, 인지부터 재구매까지의 터치포인트와 감정 곡선을 설계합니다 — pm-skills 원본 프레임워크를 FPOF 패션 하우스 컨텍스트에 맞게 커스터마이징

## 언제 사용
- 시즌 캠페인의 고객 경험 흐름을 설계할 때
- 채널별(자사몰/무신사/29CM 등) 구매 여정을 비교 분석할 때
- 고객 이탈 포인트를 식별하고 개선 방안을 도출할 때
- PDCA Do 단계에서 IMC 전략 실행과 연계할 때

## FPOF 컨텍스트
- 이 스킬은 와키윌리 브랜드의 6개 유통채널별 고객 여정 설계에 활용합니다
- 관련 프리셋: `personas.json`(UNI/WOMAN 페르소나), `channels.json`(6개 채널), `tone-manner.json`(톤앤매너)
- 코어 타겟(18~25세)의 모바일 중심, SNS 기반 구매 여정이 핵심입니다
- IP 캐릭터(키키 등)가 터치포인트별 브랜드 경험에 어떻게 관여하는지 반영합니다

## 사전 준비
1. `.fpof-state.json` → 현재 시즌 확인
2. `personas.json` → UNI/WOMAN 페르소나, JTBD 참조
3. `channels.json` → 6개 채널별 특성 및 매출 구조 참조
4. `tone-manner.json` → 채널별 커뮤니케이션 톤 참조
5. 기존 인터뷰, 서베이, 분석 데이터가 있다면 준비

## 실행 절차

### Step 1: 페르소나 정의
여정의 주인공을 특정합니다.
- `personas.json` 기반 UNI 또는 WOMAN 페르소나 선택
- JTBD(Jobs-to-be-Done)와 핵심 동기 명확화
- 제네릭 사용자가 아닌 구체적 페르소나로 설정

### Step 2: 여정 단계 매핑
와키윌리 고객 여정의 각 단계를 정의합니다.

| 단계 | 설명 |
|------|------|
| **인지 (Awareness)** | 어떻게 와키윌리를 처음 알게 되는가? (SNS, 인플루언서, 지인 추천) |
| **탐색 (Consideration)** | 무엇을 비교하고 평가하는가? (무신사 vs 자사몰, 리뷰, 룩북) |
| **구매 (Acquisition)** | 어떻게 구매하는가? (채널 선택, 결제, 프로모션) |
| **언박싱/첫 경험 (Onboarding)** | 첫 경험은 어떤가? (패키징, 핏, 기대 대비 만족도) |
| **착용/활용 (Engagement)** | 일상에서 어떻게 활용하는가? (스타일링, SNS 공유) |
| **재구매 (Retention)** | 무엇이 다시 오게 만드는가? (새 시즌, 멤버십, 신상 알림) |
| **추천 (Advocacy)** | 언제, 왜 추천하는가? (OOTD 태그, 리뷰, 지인 추천) |

### Step 3: 단계별 상세 문서화
각 단계에서 다음을 기록합니다.
- **터치포인트**: 고객이 브랜드/상품/팀과 접촉하는 지점 (인스타그램, 무신사, 자사몰, DM, 패키지)
- **고객 행동**: 이 단계에서 하는 행동
- **생각 & 질문**: 머릿속에 떠오르는 것 ("이 브랜드 뭐야?", "사이즈 괜찮을까?")
- **감정**: 느끼는 감정 (설렘, 혼란, 불안, 만족) — 스케일 또는 감정 지표
- **페인 포인트**: 마찰, 혼란, 이탈 리스크
- **기회**: 경험을 개선할 수 있는 지점

### Step 4: 크리티컬 모먼트 식별
핵심 결정적 순간을 찾습니다.
- **Aha Moment**: 와키윌리의 핵심 가치를 처음 경험하는 순간
- **Moment of Truth**: 구매 결정 또는 이탈을 결정짓는 순간
- **Churn Trigger**: 고객이 가장 많이 이탈하는 지점

### Step 5: 채널별 여정 비교
와키윌리 6개 채널의 여정 차이를 분석합니다.
- 자사몰 vs 무신사 vs 29CM 여정의 핵심 차이
- 채널별 고유 터치포인트 및 페인 포인트
- 채널 간 이동(Cross-channel) 패턴
- 각 채널의 감정 곡선 비교

### Step 6: 여정 맵 테이블 작성

| 단계 | 터치포인트 | 고객 행동 | 감정 | 페인 포인트 | 기회 |
|------|-----------|----------|------|-----------|------|

### Step 7: 우선순위 개선 권고
페인 포인트를 기준으로 개선안을 제안합니다.
- 전환율/리텐션에 가장 큰 영향을 미치는 페인 포인트
- 즉시 실행 가능한 Quick Win
- 더 큰 투자가 필요하지만 최대 효과를 내는 개선
