# IMC 전략 & GTM

> 마케팅 디렉터가 통합 마케팅 커뮤니케이션 전략을 수립하고 런칭을 총괄하는 전문 매뉴얼

## 언제 사용
- "마케팅 전략 짜줘", "캠페인 어떻게 갈까?", "GTM 계획 세워줘"
- PDCA Do 단계에서 마케팅 작업 시작 시

## 사전 준비
1. `output/[시즌]/plan/brand-strategy.md` → 시즌 테마, 타겟
2. `output/[시즌]/plan/season-concept.md` → 챔피언 상품, 가격 전략
3. `presets/wacky-willy/channels.json` → 6개 채널 매출 목표/성장률
4. `presets/wacky-willy/brand.config.json` → 경영목표 (히트상품+IMC)
5. `presets/wacky-willy/personas.json` → 타겟 미디어 소비 습관

## 캠페인 3단계 설계

**Phase 1: TEASING (D-14 ~ D-7)**
- 목적: 호기심 유발, 기대감 조성
- 채널: 인스타 스토리, TikTok 티저
- 콘텐츠: 실루엣 힌트, 컬러 티저, 카운트다운
- KPI: 도달율, 저장/공유 수

**Phase 2: MAIN LAUNCH (D-Day ~ D+7)**
- 목적: 전면 공개, 구매 전환
- 채널: 전 채널 동시
- 콘텐츠: 룩북, 화보, PDP, 인플루언서 리뷰
- KPI: 매출, 전환율, ROAS

**Phase 3: SUSTAIN (D+7 ~ D+30)**
- 목적: 지속 노출, 리뷰/UGC 확산
- 채널: SNS, 자사몰 기획전
- 콘텐츠: 착용샷, 스타일링 가이드, 고객 리뷰
- KPI: 재구매율, UGC 수, 팬덤 활성도

## 채널 믹스 설계
channels.json 기반으로 채널별 역할과 예산을 배분한다.
- 온라인: 자사몰(전환), 외부몰(노출), SNS(인게이지먼트)
- 오프라인: 샵인샵(볼륨), FSS(경험), 면세(글로벌)
- 글로벌: 대만 1호점 연계 (경영목표 #5)

## GTM 전략 프레임워크 (PM-Skills 통합)
채널별 적합성을 체계적으로 평가하고 메시지-마켓 핏을 설계한다.

### 채널 적합성 평가
channels.json의 6개 채널을 아래 기준으로 평가한다:
- **타겟 도달력:** 코어 타겟(18~25세)이 해당 채널에 존재하는가?
- **메시지 전달력:** 와키윌리의 브랜드 보이스를 효과적으로 전달할 수 있는가?
- **전환 잠재력:** 인지 → 관심 → 구매 전환이 자연스러운 채널인가?
- **비용 효율성:** ROAS 기대치가 적정한가?
- **측정 가능성:** KPI를 정확히 트래킹할 수 있는가?

| 채널 | 도달력 | 메시지 전달 | 전환 잠재 | 비용 효율 | 측정 가능 | 총점 | 우선순위 |
|------|--------|-----------|----------|----------|----------|------|---------|

### 메시지-마켓 핏 설계
- 핵심 가치 제안 (코어 타겟 세그먼트별)
- 핵심 차별화 포인트 (경쟁사 대비)
- 페인 포인트 검증 및 솔루션 매핑
- 채널별 메시지 변형 (tone-manner.json 기준)
- 소셜 프루프 전략 (인플루언서, UGC, 리뷰)

### 런칭 타임라인 (90일 로드맵)
| 기간 | 단계 | 핵심 액션 | 성공 기준 | Go/No-Go 판단 |
|------|------|----------|----------|-------------|

## GTM 모션 평가 (PM-Skills 통합)
7가지 GTM 모션을 와키윌리 맥락에서 스코어링(1~10)하여 최적 조합을 선정한다.

| # | GTM 모션 | 설명 | 와키윌리 적합도 (1-10) | 활용 방안 | 우선순위 |
|---|---------|------|---------------------|----------|---------|
| 1 | **Inbound** | 콘텐츠/SNS를 통한 자연 유입 | | 룩북, 스타일링 콘텐츠 | |
| 2 | **Outbound** | 직접 아웃리치 (DM, 이메일) | | VIP 고객 다이렉트 | |
| 3 | **Paid Digital** | 유료 광고 (메타, 구글, TikTok) | | 챔피언 상품 집중 부스팅 | |
| 4 | **Community** | 팬덤/커뮤니티 기반 성장 | | 디스코드, 오픈채팅 | |
| 5 | **Partners** | 파트너십/콜라보 마케팅 | | 타 브랜드/IP 콜라보 | |
| 6 | **ABM** | 주요 계정 집중 마케팅 | | 핵심 셀럽/KOL 맞춤 | |
| 7 | **PLG** | 제품 자체로 성장 유도 | | 한정판/굿즈 → 입소문 | |

**모션 스택 설계:**
- Primary 모션 (1순위): 가장 높은 적합도의 모션
- Secondary 모션 (2~3순위): 보완적 채널
- 모션 간 시퀀싱 (어떤 순서로 실행?)
- 리소스 배분 (예산/인력)

## 마케팅 아이디어 브레인스토밍 (PM-Skills 통합)
비용 효율적이고 창의적인 마케팅 아이디어 5개를 도출한다.

각 아이디어는:
1. **채널:** 활용할 주요 마케팅 채널 (SNS, 콘텐츠, 파트너십, 커뮤니티, 이벤트 등)
2. **핵심 메시지:** 코어 타겟에게 공명하는 메시지 (tone-manner.json 기준)
3. **왜 효과적인가:** 타겟 오디언스를 인게이지시키는 이유
4. **비용 효율성:** 제한된 예산으로 높은 임팩트를 내는 방법
5. **와키윌리 DNA 연결:** Kitsch Street & IP Universe 컨셉과의 접점

| # | 아이디어 | 채널 | 핵심 메시지 | 비용 효율 | IP 연결 |
|---|---------|------|-----------|----------|--------|
