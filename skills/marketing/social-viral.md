## 인플루언서 매핑

**매핑 기준:**
- 타겟 페르소나: UNI(남녀 공통) / WOMAN
- 팔로워 규모: 나노(1K~10K) / 마이크로(10K~50K) / 미드(50K~200K) / 매크로(200K+)
- 카테고리: 패션 / 라이프스타일 / 뷰티 / 엔터
- 브랜드핏: 와키윌리 감성(키치/스트릿/자유분방)과의 정합성

## 시딩 전략
- 시딩 상품 선정 (챔피언 상품 우선)
- 발송 시기: Teasing 단계 전
- 언박싱 가이드라인: 브랜드 경험 설계
- 콘텐츠 가이드라인: 자유도 vs 브랜드 메시지 밸런스

## 바이럴 메커니즘

- UGC 유도: 착용샷 이벤트, 스타일링 대결
- 밈 소재: 키키 캐릭터 밈 템플릿
- 커뮤니티: 팬덤 공간 (디스코드/오픈채팅)

## 그로스 루프 설계 (PM-Skills 통합)
유료 광고 의존도를 줄이고 자기 강화적(self-reinforcing) 성장 메커니즘을 설계한다.

### 5가지 그로스 루프 유형 평가
와키윌리 맥락에서 각 루프의 적합성을 평가한다:

| # | 루프 유형 | 메커니즘 | 와키윌리 적용 예시 | 적합도 (1-10) |
|---|---------|---------|-----------------|-------------|
| 1 | **Viral Loop** | 콘텐츠 생성 → 외부 공유 → 신규 유입 | 착용샷/스타일링 공유 → 인스타/TikTok 노출 → 브랜드 발견 | |
| 2 | **Usage Loop** | 사용 → 콘텐츠 생성 → 공유 → 재참여 | 키키 캐릭터 커스텀 → 공유 → 타인 참여 | |
| 3 | **Collaboration Loop** | 초대 → 공동 참여 → 가치 발견 | 친구 태그 이벤트 → 함께 쇼핑 → 공동 구매 | |
| 4 | **UGC Loop** | 발견 → 자체 콘텐츠 생성 → 공유 → 타인 발견 | 와키윌리 OOTD → TikTok 트렌드 → 모방 콘텐츠 | |
| 5 | **Referral Loop** | 추천 → 가입 → 보상 → 추가 추천 | 친구 추천 시 한정 굿즈/할인 → 추천 확산 | |

### 우선 루프 설계
가장 적합한 루프 1~2개를 선정하여 상세 설계한다:

**루프 메커니즘:**
- 트리거: 공유/초대를 시작하는 계기
- 인센티브: 참여 동기 (내재적/외재적)
- 공유 용이성: 공유 경로의 마찰도
- 전환율: 초대 → 활성 사용자 전환율 추정

**루프 계수 계산:**
- 사용자당 공유/초대 수 (사이클당)
- 초대 → 신규 사용자 전환율
- 사이클당 순 신규 사용자 수
- 사이클 소요 시간

**30-60-90일 실행 계획:**
| 기간 | 핵심 액션 | 목표 지표 | 성공 기준 |
|------|----------|----------|----------|
| 30일 | 루프 MVP 구현, 초기 테스트 | | |
| 60일 | 메시지/마찰 최적화, 전환율 개선 | | |
| 90일 | 스케일링, 2차 루프 추가 | | |
