# 카피라이팅 & 스토리텔링

> 패션 에디터가 브랜드 보이스로 모든 텍스트 콘텐츠를 작성하는 전문 매뉴얼

## 언제 사용
- "상품 설명 써줘", "캡션 만들어줘", "스토리 정리해줘"
- Do 단계에서 상품/캠페인 카피 작업 시

## 사전 준비 (필수!)
1. `presets/wacky-willy/tone-manner.json` → **반드시 참조** (모든 카피의 톤 기준)
2. `presets/wacky-willy/ip-bible.json` → 캐릭터 스토리 연결 시
3. `presets/wacky-willy/personas.json` → 타겟 언어/감성
4. `output/[시즌]/design/[아이템]-design-spec.md` → 상품 정보

## 톤앤매너 사전 체크 (매번 확인!)
- [ ] 유쾌하고 자유분방한 톤? (경직된 어투 X)
- [ ] Gen-Z 친화적 언어? (과도한 존댓말 X)
- [ ] 브랜드 필수 어휘 포함? (와키윌리, KIKY 등)
- [ ] 금지 어휘 미사용? (저렴한, 짝퉁, 따라하기, 평범한)
- [ ] 채널에 맞는 톤과 길이?

## PDP 카피 구성
1. **상품명** (국문/영문) — 브랜드 네이밍 룰 적용
2. **한 줄 후크** — 스크롤을 멈추게 하는 첫 문장
3. **상품 스토리** — 3~5문장, 시즌 테마/IP 세계관과 연결
4. **핵심 셀링 포인트** — 3개, 기능+감성 밸런스
5. **소재/케어 정보**
6. **사이즈 가이드 한 줄**
7. **추천 스타일링 팁**

## SNS 채널별 카피

**인스타그램 피드:**
- 캡션: 감성적, 2~3줄
- 해시태그: 한글+영문 믹스, 10~15개
- CTA: 좋아요/댓글/저장 유도

**TikTok:**
- 텍스트 오버레이: 1~2줄, 임팩트
- 캡션: 짧게, 참여 유도
- 해시태그: 트렌드 태그 포함, 5~8개

**자사몰 배너:**
- 헤드라인: 1줄, 강렬하게
- 서브카피: 1줄, 부연 설명
- CTA 버튼 문구: 행동 유도

## 밸류 프로포지션 스테이트먼트 (PM-Skills 통합)
타겟 세그먼트별로 와키윌리의 가치 제안을 명확한 카피로 작성한다.

### VP 스테이트먼트 생성 프레임
각 타겟 세그먼트(personas.json 기준)에 대해 아래 구조로 VP 카피를 작성한다:
- **타겟 세그먼트:** 특정 타겟 또는 유스케이스를 명시
- **핵심 베네핏:** 원하는 결과/감정을 강조
- **핵심 기능/역량:** 그것을 가능하게 하는 와키윌리만의 요소
- **명확하고 공감 가는 언어:** tone-manner.json 기준 준수

### 용도별 VP 스테이트먼트

**마케팅용 (광고, SNS, 배너):**
- UNI 타겟 VP: "와키윌리는 [타겟]을 위한 [핵심 베네핏]을 제공하는 유일한 [카테고리]이다. [서포팅 메시지]"
- WOMAN 타겟 VP: (동일 구조)

**세일즈용 (바이어 미팅, B2B 프레젠테이션):**
- 유통 바이어 대상 VP: 매출 기여도, 셀스루율, 브랜드 화제성 중심
- 해외 바이어 대상 VP: K-컬처, IP 유니버스, 글로벌 확장성 중심

**온보딩용 (신규 고객 첫 접점):**
- 자사몰 첫 방문 VP: 브랜드 세계관 소개 + 첫 구매 동기 부여
- 팬덤 커뮤니티 가입 VP: 소속감, 한정판 혜택, 내부자 경험

### VP 품질 체크
각 VP 스테이트먼트는:
- [ ] 특정 타겟 세그먼트를 직접 언급하는가?
- [ ] 핵심 베네핏과 원하는 결과를 강조하는가?
- [ ] 그것을 가능하게 하는 핵심 기능/역량을 포함하는가?
- [ ] tone-manner.json의 브랜드 보이스와 일치하는가?
- [ ] 경쟁사와 차별화되는 포인트가 있는가?
