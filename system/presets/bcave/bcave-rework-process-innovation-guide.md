# 비케이브 전사 방향성 가이드

## 한 문장 요약

> **"AI 시대, 뾰족한 브랜드와 빠른 의사결정으로 생존하고 성장한다."**

2026년 비케이브는 **기능 중심 → 브랜드 중심** 조직으로 전환하고, **AI 기반 업무 혁신**과 **브랜드 감도 강화**를 양대 축으로 전사적 체질 개선을 실행합니다.

---

## 1. 핵심 변화 방향 — 5대 전환

| # | FROM (기존) | TO (전환) | 핵심 메시지 |
|---|-----------|---------|-----------|
| 1 | 기능 조직 (기획·디자인·영업·물류) | **브랜드 중심 조직** | 브랜드 책임자가 PMD(상품·마케팅·유통) 방향성 통합 관리 |
| 2 | AI = 효율 개선 도구 | **AI = 조직을 다시 짜는 것** | 이번 변화는 선택이 아닌 **생존** |
| 3 | 안전하고 무난한 상품·콘텐츠 | **뾰족하고 감도 높은 브랜드** | SPA와 차별화 없이는 생존 불가 |
| 4 | 면세·할인 의존 유통 | **해외 직진출 + 내수 손익 관리** | 면세 점진적 축소, B2C 전환 |
| 5 | 분절적 상품 기획 | **히트상품 원팀 발굴 프로세스** | 영업·마케팅·브랜드가 원팀으로 리소스 집중 |

## 2. 브랜드 감도 강화 — 모든 접점에서 뾰족하게

### 2-1. 왜 감도인가?

- SPA(무신사 스탠다드, 유니클로)의 영역 확장으로 **차별화 없이는 생존 불가**
- 상품 70점을 마케팅이 90점으로 올려야 하나, 현재 두 가지 모두 미흡
- 문제는 특정 브랜드가 아닌 **전사적 공통 과제**

### 2-2. 영역별 전환 방향

| 영역 | 현재 문제 | 전환 방향 |
|------|----------|----------|
| **상품** | 너무 안전하고 무난함. 매장 구성이 SPA와 차별성 없음 | 일부 카테고리는 **강제로 뾰족한 상품 편성**. 실패 감수하고 도전 |
| **마케팅** | 안정 지향적 콘텐츠. 기억에 남지 않고 화제성 부족 | 논란 가능성 있어도 **트렌드 따라 밀어붙이기**. 반응 있는 콘텐츠 우선 |
| **스태프 스타일링** | 유니폼처럼 획일적 | 아카이브·여성·트렌드 라인별 **다양화** |
| **콘텐츠** | 셀럽 의존, 브랜드 자체 서사 부족 | 브랜드 자체 **크루·커뮤니티·로컬 기반 세계관** 구축. 셀럽은 확성기 역할 |
| **속도** | 헤비한 콘텐츠에 시간 소모 | 빠르게 쪼개서 **스피디하게 실행** |

### 2-3. 브랜드별 핵심 피드백

- **리(Lee)**: 유니 룩북의 모범생 탈피, 트렌드 룩북에 유행 반영 강화
- **와키윌리**: 아이돌이 입을 만한 트렌디한 상품 확대, 과감한 소재·디자인 시도
- **커버낫**: TNG 개선 유지, 유니 라인 스타일링 전면 개선
- **마케팅 전반**: 셀럽 제외 시에도 브랜드 자체 서사·세계관·커뮤니티 콘텐츠 확보

### 2-4. 대표 직속 감도 체크 조직 신설

- **배경**: 20대 초중반 타겟 브랜드의 감도를 경영진이 직접 판단하는 것은 한계
- **구성**: 1~2명 규모 대표 직속 조직
- **역할**: 상품·마케팅·스타일 감도 피드백 및 옵션 제시
- **원칙**: 라인 조직은 무조건 수용이 아닌, **옵션을 듣고 자체 판단·의사결정 후 실행**

## 3. AI 기반 조직 변혁 — "판을 설계하는 시대"

### 3-1. 대표 핵심 메시지

> [!quote]
> "AI 시대는 일 많이 하는 사람의 시대가 아니라, **판을 설계하고 선택하고 책임지는 사람**의 시대. ERP 완성까지 기다리자는 6~8개월 지연이 아니라, 다른 회사 대비 7~8년이 늦어지는 것."

### 3-2. 사람만이 해야 할 일 vs. AI 대체 영역

**사람의 영역 (대체 불가)**:

| 역할 | 설명 |
|------|------|
| **전략적 의사결정** | 무엇을 할지/말지, 어떤 브랜드를 키울지, 어떤 사업을 그만둘지 |
| **리스크 있는 판단** | 책임 있는 의사결정. 늦은 결정 = 한 달 단위 기회 손실 |
| **사람을 움직이는 일** | 조직 관리, 동기부여, 협업 촉진 |
| **창의력·감각·직관** | 트렌드 감지, 브랜드 감도, 시스템 설계 |

**AI로 대체·압축 가능한 영역**:

| 영역 | 구체적 업무 |
|------|-----------|
| 디자인실 | 레퍼런스 수집, 컬러 변형, 그래픽 작업 |
| 물류 | 운영 최적화 (즉시 가능) |
| 경영기획/전략실 | 레포트 생성, 데이터 분석 |
| 마케팅 | 단순 SNS·카피 생성, 트렌드 분석 자동화 |
| 기획·소싱 | 시즌 반복·레퍼런스 수집 자동화 |

### 3-3. AI 시대에 귀해질 인재

- 브랜드 정체성을 만들 수 있는 **디자이너**
- 트렌드를 읽고 **크리에이터와 협업**할 수 있는 사람
- PMD(상품·마케팅·유통) 방향성을 이해하고 관리할 수 있는 **키맨**
- AI 활용 기반으로 업무를 **설계하고 실행**시킬 수 있는 사람
- 유통 채널에 대한 **깊은 이해**를 가진 사람

### 3-4. 관리자·팀원 필수 역량

**관리자 필수 역량 5가지**:
1. 업무 프로세스 최신화
2. 조직관리·성과관리
3. 감도 판단력
4. 유관부서 커뮤니케이션
5. 상업적 감각

**팀원 필수 역량**: 감도, 실행력, 속도, AI 활용 능력

> [!important] 트렌드 민감형 인재 배치
> 각 조직에 **트렌드 민감형 인재**(릴스·쇼츠 중독 수준) 반드시 1명 배치.
> 관리자는 그 아이디어 판단·실행 지원. 감도 약한 관리자는 감도 높은 짝꿍 배치로 보완.

### 3-5. AI 전환 로드맵 (6개월)

| 기간 | 내용 |
|------|------|
| **1~2개월** (4~5월) | 자동화 가능 업무 식별, 조직 재점검, AX TF 구성 |
| **3~4개월** (6~7월) | ERP·데이터 연계 업무 방식 설계, 파일럿 과제 실행 |
| **5~6개월** (8~9월) | 조직 재배치, 인력 재배치, 성과 평가 기준 전면 재설계 |

### 3-6. AX(AI Transformation) 체계

**AI 도입 5단계**:

| 단계  | 내용                      | 현재 상태   |
| --- | ----------------------- | ------- |
| 1단계 | 생성형 AI 기본 활용 (보고서, 회의록) | ✅ 완료    |
| 2단계 | 보안/감사 통제 하 체계적 제공       | 🔄 진행 중 |
| 3단계 | 파일럿 검증 → 성과 평가          | 📋 계획   |
| 4단계 | 거버넌스 심사 후 전사 확산         | 📋 계획   |
| 5단계 | 인프라 투자 + 조직 확대          | 📋 계획   |

**AX 운영 원칙**:
- 현업이 만들고, IT가 보완
- **ERP/WMS 등 원천 데이터 변경·훼손은 AI 금지**
- AI/로코딩은 데이터 분석·업무 개선 용도로 제한
- 모든 결과물은 정기 심사 대상
- AI 생성 데이터는 개인 PC가 아닌 **IT 인프라 종속 DB에 집중** → 전사 공유

**거버넌스 프로세스**:

```
현업 TF 8~10명 (바이브코딩 → 프로토타입/POC)
    ↓
AX 거버넌스 커미티 심사 (경영진 참여, 격주/월)
    ↓
┌─ 확산: RPA → 서버 배포 / 현황판 → 사내 플랫폼(EIS)
├─ 홀딩: 요건 재정비 → 재심사
└─ 중단: 실패 보고서 작성 → 레퍼런스 축적
```

## 4. 조직 재설계 — 기능 조직에서 브랜드 중심 조직으로

### 4-1. 전환 구조

```
[기존] 기능 조직                    [전환] 브랜드 중심 조직
─────────────────                ─────────────────────
기획팀 ─┐                        브랜드A 책임자
디자인팀 ├→ 전 브랜드 지원            ├─ 영업 담당
영업팀 ─┤                          ├─ 마케팅 담당
물류팀 ─┘                          └─ PMD 방향성 통합 관리

                                 브랜드B 책임자
                                   ├─ ...
                                   └─ ...

                                 IT팀 (강화)
                                   ├─ ERP/데이터
                                   └─ AI 일하는 방식
```

### 4-2. 핵심 원칙

- 브랜드 책임자가 **영업·마케팅 담당자와 브랜드 방향성을 직접 공유**
- 각 브랜드·영업 내 **챔피언·키맨 육성 및 강화**
- IT팀은 ERP, 데이터, AI 일하는 방식을 전사 지원
- 인사평가 기준 전면 재설계 (AI 역량, 감도, 실행력 반영)

### 4-3. 문화 전환 원칙

> [!quote] 대표 메시지
> "문제는 사람이 아니라 **시스템** — 의사결정 구조·프로세스·리스크 회피 문화를 개선하자."

- **의사결정 속도**: 늦은 결정 = 한 달의 기회 손실. 바텀업 구조가 핵심
- **데이터 기반**: 감각과 데이터를 함께 놓고 판단
- **리스크 수용**: 실패 감수하고 도전하는 문화. 리스크 회피 문화 탈피
- **실무단 즉시 실행**: 실무단 + 책임자급이 운영하고 바로 결정·실행

## 5. 유통 전략 재편

### 5-1. 유통 전략 3원칙

| 우선순위 | 원칙 | 설명 |
|---------|------|------|
| 1순위 | **내수 손익 관리** | 적자 매장 정리, BP 매출 회복 |
| 2순위 | **해외 B2C 매출 확보** | 중국·대만·일본 직진출 |
| 3순위 | **B2B 업체 확보** → 이후 면세 축소 | 대체 매출 확보 후 면세 전환 |

### 5-2. 면세 채널 — 점진적 축소

- 해외 직진출 타임라인에 맞춰 면세 → B2C 전환
- 현지 법인 설립 → 내수 진입 환경 구축 우선
- 기존 면세 계약(잔여 10개월) 유지하되 개런티 확보, 수수료·관리비 인하 협상 지속

### 5-3. 리테일 매장 구조조정

| 유형             | 대상                                   | 대응 방안                          |
| -------------- | ------------------------------------ | ------------------------------ |
| **활성화** (5개)   | 스타필드 고양(커버낫, 와키윌리), 스타필드 수원·부천, 리 명동 | BP 매출 회복, 프로모션 강화, 라이브커머스      |
| **구조 변경** (3개) | 스타필드 인테리어 리뉴얼 대상, 리 홍대 FIT 특화        | 인테리어 리뉴얼, 인바운드 특화 전환, 도급사 변경   |
| **폐점** (1개+)   | 여주 아울렛, 현대면세점 무역센터                   | 조기 폐점 또는 통합 운영, 면세 인력 플래그십 재배치 |


### 5-4. 플래그십 출점 원칙

- 동일 에어리어 추가 오픈 **제한**
- 향후 오픈 가능 지역: **한남, 북촌, 성수, 서촌** 또는 신규 상권만

## 6. 히트상품 발굴 프로세스 혁신

### 6-1. 핵심 관점 전환

> [!quote] 대표 메시지
> "몇 장 팔았느냐보다 **히트상품을 발굴하기 위해 일하는 방식이 어떻게 바뀌었느냐**가 더 중요하다."

### 6-2. 기존 vs. 신규 프로세스

```
[기존] 분절적 프로세스
시즌 기획 금액 → 메인/QR 구분 → 상품 배치 → 마케팅 (순차적·분절적)

[전환] 원팀 프로세스
유효 스타일 발굴 → 의사결정 근거 기록
    ↓
VM·마케팅·영업 연계 → 리소스 집중 투입 (원팀)
    ↓
인플루언서 시딩 → 빠른 실행
    ↓
모니터링 & 의사결정 (실무단 + 책임자급 즉시 결정)
```

### 6-3. 모니터링 체계

- **1시트**: 브랜드별 IMC 주력 스타일 현황
- **2시트**: 리오더 스타일 체크업
- **대시보드**: 마케팅 투자 대비 효율 측정, 씨딩 결과 측정
- **인플루언서 그루핑**: 크루·셀럽·마이크로 → 스타일별 매칭

### 6-4. 무신사 전략 재정립

- **부진의 본질**: 할인 프로모션 부족이 아닌, **크리에이터×브랜드×상품 삼각 협업 부재**
- 인플루언서가 **제작 시점부터 관여** → 영상 제작 → 채널 광고
- 소규모 회사 벤치마크: 직원 3~5명, AI + 크리에이터 활용으로 매출 150~200억

## 8. 업무 시 참고 체크리스트

### 의사결정 전 자문 5가지

- [ ] 이 결정이 **브랜드 감도**를 높이는가?
- [ ] **AI로 대체 가능한 업무**에 사람 리소스를 쓰고 있지 않은가?
- [ ] **데이터 근거**가 있는 판단인가?
- [ ] **속도**가 충분한가? (늦은 결정 = 한 달의 기회 손실)
- [ ] **손익**을 함께 고려했는가? (매출만이 아닌 이익·재고까지)

### 각 직급별 핵심 행동 기준

| 직급 | 핵심 행동 |
|------|----------|
| **경영진** | 빠른 의사결정, 데이터 해석, 시스템 설계, 인재밀도 관리 |
| **관리자** | 프로세스 최신화, 조직관리, 감도 판단, 유관부서 소통, 상업적 감각 |
| **팀원** | 감도, 실행력, 속도, AI 활용 능력 |
