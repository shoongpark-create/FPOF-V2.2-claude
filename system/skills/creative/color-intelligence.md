# 컬러 인텔리전스

> 브랜드 시그니처 컬러를 기반으로 트렌드 컬러, 경쟁사 컬러, 소비자 선호를 분석하여
> 시즌 마스터 팔레트와 카테고리별 컬러 배분을 설계하는 컬러 전략 스킬.

## 기존 스킬과의 관계

| 기존 스킬 | 역할 | color-intelligence에서의 위치 |
|-----------|------|---------------------------|
| `moodboard` | 시즌 컬러 팔레트 구성 (수동, 정성적) | Step 2 무드보드의 컬러 섹션을 데이터로 강화 |
| `design-spec` | 아이템별 컬러웨이 선정 | Step 5 결과가 design-spec의 컬러 입력으로 연결 |
| `trend-radar` | 비주얼 시그널 중 컬러 트렌드 | Step 2 트렌드 컬러 데이터 피드 |
| `design-generator` | 컬러웨이 매트릭스 | Step 5 결과가 design-generator의 컬러 입력 |

**color-intelligence = 무드보드의 "컬러 팔레트 구성"을 독립 + 정량화한 전문 스킬**

## 언제 사용

- "이번 시즌 컬러 어떻게 가져갈지 정리해줘" — 시즌 킥오프 시
- "컬러웨이 어떻게 짜지?" — 아이템 디자인 전 컬러 전략 필요 시
- "경쟁사 컬러 분석해줘" — 무신사 랭킹 데이터 기반 컬러 분석
- Plan 단계 (trend-radar 후) 또는 Design 단계 (moodboard 전/동시)

## 사전 준비

1. `presets/wacky-willy/visual-identity.json` → 시그니처 컬러, 그래픽 스타일
2. `presets/wacky-willy/categories.json` → 라인별 히어로 아이템
3. `presets/wacky-willy/brand.config.json` → 포지셔닝 (UNI/W 감성 차이)
4. trend-radar 결과 (있으면) → 비주얼 시그널 중 컬러 데이터
5. 무신사 랭킹 데이터 (있으면) → 경쟁사 컬러 분석

## 실행 절차

### Step 1: 브랜드 컬러 DNA 로드

`visual-identity.json`에서 변하지 않는 브랜드 컬러 기반을 확인한다.

```
브랜드 시그니처 (불변):
  Primary:
    ■ Signature Yellow  #FEF200  — 브랜드 아이덴티티
    ■ Black             #000000  — 기본
    ■ White             #FFFFFF  — 기본
  Secondary:
    ■ Sky Blue          #68A8DB  — 시즌 악센트

사용 원칙:
  - Primary = 브랜드 아이덴티티 (Black + Signature Yellow)
  - Secondary = 시즌 악센트
  - 시그니처 컬러는 매 시즌 최소 1개 히어로 아이템에 반드시 적용
```

---

### Step 2: 트렌드 컬러 수집

글로벌 컬러 트렌드와 패션 시장의 컬러 동향을 수집한다.

**수집 소스:**

| 소스 | 수집 방법 | 추출 데이터 |
|------|----------|-----------|
| PANTONE COTY | 웹서치 "Pantone Color of the Year 2026" | 올해의 컬러 + 팔레트 |
| PANTONE 시즌 리포트 | 웹서치 "Pantone fashion color trend [시즌]" | 시즌별 10~12색 팔레트 |
| 런웨이 컬러 | 웹서치 "runway color trend [시즌]" | 반복 등장 컬러, 신흥 컬러 |
| 스트릿 컬러 | 웹서치 "street style color trend" | 실제 착용 인기 컬러 |
| SNS 컬러 | 웹서치 "TikTok fashion color popular" | Z세대 선호 컬러 |

**추출 결과:**
```markdown
| # | 트렌드 컬러명 | HEX 근사 | 소스 | 강도 | 와키윌리 적합도 |
|---|-------------|---------|------|------|---------------|
```

---

### Step 3: 경쟁사 컬러 스캔

무신사 랭킹/발매 데이터 또는 웹서치로 경쟁 브랜드의 컬러 전략을 분석한다.

**분석 항목:**

| 항목 | 방법 |
|------|------|
| 경쟁사 메인 컬러 | 상위 판매 아이템의 컬러 분포 |
| 카테고리별 인기 컬러 | 상의/아우터/하의별 판매 상위 컬러 |
| 가격대별 컬러 | 엔트리/미들/프리미엄 컬러 차이 |
| 공백 컬러 | 경쟁사가 쓰지 않는 컬러 → 차별화 기회 |

**경쟁사 컬러 매트릭스:**
```markdown
| 브랜드 | 메인 컬러 | 서브 컬러 | 시즌 특이 컬러 | 공백 |
|--------|---------|---------|-------------|------|
```

---

### Step 4: 시즌 마스터 팔레트 설계

브랜드 DNA + 트렌드 + 경쟁사 분석을 종합하여 시즌 팔레트를 구성한다.

#### 팔레트 구조 (12~16색)

```
┌─────────────────────────────────────────────┐
│  CORE (4색) — 매 시즌 불변                    │
│  ■ #FEF200  ■ #000000  ■ #FFFFFF  ■ #68A8DB │
├─────────────────────────────────────────────┤
│  SEASON MAIN (4~6색) — 시즌 메인 컬러         │
│  ■ [색1]  ■ [색2]  ■ [색3]  ■ [색4]          │
│  선정 기준: 트렌드 강도 + 브랜드 적합도 + 상업성│
├─────────────────────────────────────────────┤
│  ACCENT (2~3색) — 실험/화제성 컬러             │
│  ■ [색1]  ■ [색2]                             │
│  선정 기준: 경쟁사 공백 + 브랜드 DNA 매칭       │
├─────────────────────────────────────────────┤
│  NEUTRAL (2~3색) — 베이스/조합용               │
│  ■ [색1]  ■ [색2]                             │
│  선정 기준: 코어 컬러와 조화, 스타일링 기반     │
└─────────────────────────────────────────────┘
```

#### 컬러 스코어링 (선정 근거)

각 후보 컬러를 4축으로 평가:

| 축 | 기준 | 1점 | 10점 |
|---|------|-----|------|
| **Trend** | 트렌드 적합도 | 쇠퇴기 | PANTONE COTY/SNS 급상승 |
| **Brand** | 와키윌리 DNA 매칭 | 컨셉 충돌 | Kitsch/Street/Vivid 완벽 |
| **Commercial** | 상업적 판매 가능성 | 니치/실험적 | 대중 선호, 판매 증명 |
| **Whitespace** | 경쟁사 비사용 | 레드오션 컬러 | 차별화 가능 |

```markdown
| 후보 컬러 | HEX | Trend | Brand | Commercial | Whitespace | 종합 | 배정 |
|----------|-----|-------|-------|-----------|-----------|------|------|
```

---

### Step 5: 카테고리별 컬러 배분

마스터 팔레트를 라인/카테고리/BTA별로 배분한다.

#### 라인별 컬러 성격

| 라인 | 컬러 성격 | Core 활용 | Season 활용 | Accent 활용 |
|------|----------|----------|-----------|-----------|
| **UNI** | 볼드, 비비드, 하이콘트라스트 | Yellow+Black 적극 | 비비드 시즌 컬러 | 네온/특수 컬러 |
| **W** | 페미닌, 파스텔+포인트, 세련 | White+Sky Blue | 파스텔/뮤트 톤 | 유니크 포인트 |
| **ACC** | 아이코닉, 캐릭터 시그니처 | Yellow+Black 핵심 | 캐릭터별 시그니처 | 한정판 컬러 |

#### BTA × 컬러 매핑

| BTA | 컬러 전략 | 컬러 종류 | 리오더 가능성 |
|-----|----------|----------|-------------|
| **Basic** | 안전 컬러 — 블랙, 화이트, 그레이, 네이비 | Neutral + Core | 높음 (정기 리오더) |
| **Trend** | 시즌 컬러 — 트렌드 반영, 상업성 증명 | Season Main | 중간 (히트 시 리오더) |
| **Accent** | 실험 컬러 — 차별화, 화제성, IP 특별판 | Accent | 낮음 (한정 생산) |

#### 카테고리별 컬러 배분 테이블

```markdown
| 카테고리 | 라인 | BTA 비중 | Core | Season | Accent | Neutral | 총 컬러웨이 |
|---------|------|---------|------|--------|--------|---------|-----------|
| 그래픽 티 | UNI | T70 A20 B10 | 2 | 3 | 1 | 1 | 7 |
| 스웻/후드 | UNI | T50 B30 A20 | 2 | 2 | 1 | 2 | 7 |
| 크롭 가디건 | W | T60 A30 B10 | 1 | 3 | 1 | 1 | 6 |
| 용품 | ACC | A40 T40 B20 | 2 | 2 | 2 | 0 | 6 |
```

---

### Step 6: 컬러 스토리 작성

팔레트에 시즌 내러티브를 부여한다.

```markdown
## [시즌] 컬러 스토리: [테마명]

[3~5문장의 시즌 컬러 내러티브]
— 브랜드 DNA와 시즌 테마를 연결하는 감성적 설명
— 무드보드, 프레젠테이션, 바이어 미팅에서 활용

### 컬러 그룹별 스토리
- **Core**: [와키윌리의 변하지 않는 정체성 — Yellow의 에너지, Black의 엣지]
- **Season Main**: [이번 시즌 키워드와 연결 — 왜 이 컬러가 이 시즌인지]
- **Accent**: [차별화 포인트 — 왜 이 컬러가 실험적 가치가 있는지]
- **Neutral**: [스타일링 기반 — 메인 컬러를 돋보이게 하는 역할]
```
