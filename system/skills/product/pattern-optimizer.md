# 패턴 최적화

> 아이템의 패턴 구성을 분석하여 원단 사용량(용척)을 최적화하고,
> 대안 소재 제안과 원가 절감 시뮬레이션을 수행하는 생산 효율화 스킬.
> costing-ve의 "소재 VE"를 데이터 기반으로 고도화한 전문 스킬.

## 기존 스킬과의 관계

| 기존 스킬 | 역할 | pattern-optimizer에서의 위치 |
|-----------|------|---------------------------|
| `costing-ve` | 원가 계산 + VE 제안 (전체) | Step 1~3 결과가 costing-ve의 소재비 입력으로 연결 |
| `design-spec` | 아이템 사양 (소재, 디테일) | Step 1 소재/디테일 정보 소스 |
| `techpack` | 생산용 사양서 | Step 4 결과가 techpack의 BOM 입력으로 연결 |

**pattern-optimizer = costing-ve의 "소재 VE 단계"를 독립 + 심화한 전문 스킬**

costing-ve가 전체 원가를 다루면, pattern-optimizer는 소재/패턴에 집중하여 깊게 판다.

## 언제 사용

- "이 아이템 원단 얼마나 들어?" — 용척 계산
- "원가 줄이려면 소재 어떻게 바꿔?" — 대안 소재 매트릭스
- "마커 효율 올릴 수 있어?" — 패턴 배치 최적화
- "지속가능한 소재 대안 뭐 있어?" — 친환경 소재 전환 분석
- Design 단계에서 design-spec 완료 후, costing-ve 전 또는 동시

## 사전 준비

1. `.fpof-state.json` → 현재 시즌
2. `presets/wacky-willy/categories.json` → 카테고리별 상품 전략
3. 디자인 스펙 또는 테크팩 (있으면):
   - `output/[시즌]/design/[아이템]-design-spec.md`
   - `output/[시즌]/do/[아이템]-techpack.md`
4. 기존 원가 계산서 (있으면):
   - `output/[시즌]/design/[아이템]-costing.md`

### Step 1: 패턴 피스 분석

| 아이템 유형 | 주요 패턴 피스 | 표준 피스 수 | 난이도 |

### Step 2: 용척 계산 & 마커 효율

원단 폭과 패턴 배치를 기반으로 용척(원단 사용량)을 계산한다.

#### 용척 계산

```
기본 용척 = 패턴 총면적 / 원단 폭
실사용 용척 = 기본 용척 × (1 + 로스율)

로스율 구성:
  - 마커 로스: 패턴 배치 후 남는 원단 (목표: 15% 이하)
  - 재단 로스: 재단 과정 손실 (표준: 2~3%)
  - 불량 로스: 원단 결점 (표준: 1~2%)
  - 총 로스율 = 마커 + 재단 + 불량
```

#### 마커 효율 분석

| 항목 | 현행 | 목표 | 개선안 |
|------|------|------|--------|
| 마커 효율률 | % | ≥85% | |
| 마커 로스율 | % | ≤15% | |
| 용척 (yard/pcs) | | | |
| 원단 단가 (원/yard) | | | |
| 소재비/pcs | ₩ | | |

#### 마커 효율 개선 방안

```
① 사이즈 그레이딩 조합 최적화
   - S+L 조합 vs M+XL 조합 → 피스 간 공간 활용
   - 복수 사이즈 교차 배치로 자투리 최소화

② 원단 폭 변경 검토
   - 58" → 60" 폭 변경 시 효율 변화 시뮬레이션
   - 원단 폭별 용척 비교

③ 패턴 수정 (디자인 영향 최소)
   - 곡선 → 직선 변경 가능 부분 식별
   - 시접 폭 조정 (1.5cm → 1.0cm)

④ 잔여 원단 활용
   - 자투리로 소형 ACC 제작 (패치, 키링, 파우치)
   - 캐릭터 와펜/라벨 재단
```

### Step 3: 대안 소재 매트릭스

동등 품질 또는 기능 향상 소재를 비교 분석한다.

#### 대안 소재 평가 기준

| 축 | 기준 | 설명 |
|---|------|------|
| **Cost** | 단가 비교 | 현행 대비 절감률 |
| **Quality** | 품질 동등성 | 촉감, 내구성, 세탁성 |
| **Availability** | 조달 용이성 | MOQ, 리드타임, 공급 안정성 |
| **Sustainability** | 지속가능성 | 재생, 유기농, 인증 여부 |

#### 대안 소재 비교 테이블

```markdown
| # | 소재 | 조성 | 중량 | 단가(₩/yd) | 현행 대비 | Quality | Avail | Sustain | 총평 |
|---|------|------|------|-----------|---------|---------|-------|---------|------|
| 현행 | [현재 소재] | | | | 기준 | — | — | — | — |
| 대안1 | | | | | -₩ (-%) | ○/△/× | ○/△/× | ○/△/× | |
| 대안2 | | | | | -₩ (-%) | ○/△/× | ○/△/× | ○/△/× | |
| 대안3 | | | | | -₩ (-%) | ○/△/× | ○/△/× | ○/△/× | |
```

#### 소재별 절감 시뮬레이션

```markdown
| 소재 변경 | 단가 변화 | 용척 변화 | pcs당 절감 | 총 절감 (×발주량) |
|----------|---------|---------|----------|-----------------|
| 현행 → 대안1 | -₩ | 동일 | -₩ | -₩ (×[수량]) |
| 현행 → 대안2 | -₩ | 동일 | -₩ | -₩ (×[수량]) |
```

### Step 4: 원가 절감 종합 시뮬레이션

마커 효율 + 대안 소재 + 기타 개선안을 종합하여 절감 효과를 산출한다.

#### 절감 시나리오

| 시나리오 | 내용 | pcs당 절감 | 품질 영향 | 리스크 |
|---------|------|----------|---------|-------|
| A (보수적) | 마커 효율만 개선 | -₩ | 없음 | 낮음 |
| B (표준) | 마커 + 대안 소재 | -₩ | 최소 | 중간 |
| C (적극적) | 마커 + 소재 + 디테일 간소화 | -₩ | 일부 | 중간 |

#### 원가율 변화

```markdown
| 항목 | 현행 | 시나리오A | 시나리오B | 시나리오C |
|------|------|---------|---------|---------|
| 소재비 | ₩ | ₩ | ₩ | ₩ |
| 총 원가 | ₩ | ₩ | ₩ | ₩ |
| 원가율 | % | % | % | % |
| 판정 | Over/On/Under | | | |
```

| 자투리 활용 가능성 | 없음/있음 | | ACC 전환 가능 여부 |
