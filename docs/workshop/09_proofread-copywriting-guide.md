# 세션 9: 교정 & 브랜드 보이스 — `/proofread` + `copywriting`

> **소요시간**: 25분 (설명 10분 + 실습 15분)
> **대상**: 마케팅·콘텐츠·MD 멤버
> **목표**: 대외 콘텐츠의 문법 교정과 브랜드 톤앤매너 일관성을 AI로 검수

---

## 1. 왜 필요한가?

리워크 프로세스 가이드의 핵심 키워드: **"브랜드 감도 강화"**

| 문제 | 결과 |
|------|------|
| 카피마다 톤이 다름 | 고객이 와키윌리를 기억 못 함 |
| 문법 실수가 그대로 게시 | 브랜드 신뢰도 하락 |
| 금지 어휘 사용 | 브랜드 이미지 훼손 |
| 채널별 톤 미구분 | 인스타에 보도자료 톤, 보도자료에 인스타 톤 |

---

## 2. `/proofread` — 문법 & 흐름 체크

### 사용법

```
/proofread <텍스트 또는 파일 경로>
```

### 검수 항목

| 항목 | 설명 |
|------|------|
| **맞춤법** | 한국어·영어 맞춤법 교정 |
| **문법** | 문장 구조, 호응, 조사 |
| **흐름** | 문장 간 논리적 연결, 반복 표현 |
| **브랜드 보이스** | tone-manner.json 기반 톤 일관성 |
| **금지 어휘** | "저렴한", "짝퉁", "따라하기", "평범한" 등 |
| **한영 병기** | 전문 용어 한글(영문) 형식 |

### 실습

```
/proofread "와키윌리의 저렴한 가격으로 만나보세요.
이번 시즌은 평범한 디자인을 넘어서 새로운 시도를 했습니다.
캠프키치 후드티는 기본에 충실한 아이템입니다."
```

**예상 결과:**
```
수정 전: "와키윌리의 저렴한 가격으로 만나보세요."
수정 후: "와키윌리만의 합리적인 프라이스로 만나보세요."
이유: "저렴한"은 금지 어휘 — 브랜드 가치를 낮추는 표현

수정 전: "평범한 디자인을 넘어서"
수정 후: "기존의 룰을 깨는"
이유: "평범한"은 금지 어휘 — 와키윌리 톤은 도전적·자유분방

수정 전: "기본에 충실한 아이템"
수정 후: "스트릿의 에센셜을 새롭게 해석한 아이템"
이유: BTA 가이드라인 — Basic도 브랜드화된 표현 필요 (3B 착장 금지)
```

## 3. `copywriting` — 브랜드 카피 작성

### 트리거

```
"상품 설명 써줘"
"인스타 캡션 만들어줘"
"해시태그 추천해줘"
"컬렉션 소개 글 써줘"
```

### 톤앤매너 체크리스트 (매번 자동 확인)

- [ ] 유쾌하고 자유분방한 톤? (경직된 어투 X)
- [ ] Gen-Z 친화적 언어? (과도한 존댓말 X)
- [ ] 브랜드 필수 어휘 포함? (와키윌리, KIKY 등)
- [ ] 금지 어휘 미사용?
- [ ] 채널에 맞는 톤과 길이?

### PDP 카피 구성 (7요소)

```
1. 상품명 (국문/영문)
2. 한 줄 후크 — 스크롤을 멈추게 하는 첫 문장
3. 상품 스토리 — 3~5문장, 시즌 테마/IP 세계관 연결
4. 핵심 셀링 포인트 — 3개, 기능+감성 밸런스
5. 소재/케어 정보
6. 사이즈 가이드 한 줄
7. 추천 스타일링 팁
```

### 채널별 카피 가이드

| 채널 | 톤 | 길이 | 해시태그 |
|------|-----|------|---------|
| **인스타 피드** | 감성적 | 2~3줄 + CTA | 한글+영문 믹스 10~15개 |
| **TikTok** | 임팩트·참여유도 | 1~2줄 텍스트 오버레이 | 트렌드 태그 포함 5~8개 |
| **자사몰 배너** | 강렬+행동유도 | 헤드라인 1줄 + 서브카피 1줄 | CTA 버튼 문구 포함 |
| **무신사 PDP** | 정보+감성 | 상세 (7요소 전체) | SEO 키워드 포함 |
| **보도자료** | 프로페셔널 | 1~2페이지 | 해시태그 없음 |

### 실습

```
"캠프키치 후드티 인스타 캡션 만들어줘"
```
