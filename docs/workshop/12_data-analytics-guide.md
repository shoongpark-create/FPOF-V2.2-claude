# 세션 12: 데이터 분석 & 대시보드

> **소요시간**: 35분 (설명 10분 + 데모 15분 + 실습 10분)
> **대상**: 데이터팀·MD·경영기획
> **목표**: 매출 분석, 코호트, A/B 테스트, KPI 대시보드 설계를 실습

---

## 1. 데이터 분석 스킬 구조

```
/north-star ── 핵심 지표 정의 (가장 먼저)
      ↓
/metrics ── KPI 대시보드 설계 (NSM + 인풋 + 헬스)
      ↓
sales-analysis ── 매출 분석 (채널/카테고리/SKU)
      ↓
/cohorts ── 코호트 리텐션·재구매 분석
      ↓
/ab-test ── A/B 테스트 통계 분석
```

---

## 2. `/north-star` — 핵심 지표 정의

모든 데이터 분석의 출발점. "우리의 성공을 가장 잘 나타내는 단일 지표는 무엇인가?"

```
/north-star
```

### 산출물 구성

| 요소 | 설명 |
|------|------|
| **North Star Metric** | 고객 가치를 가장 잘 포착하는 단일 지표 |
| **검증 6기준** | 가치 전달 측정, 선행 지표, 측정 가능, 전원 이해, 실행 가능, 허영 아님 |
| **인과 체인** | 인풋 메트릭 → NSM → 비즈니스 성과 연결 |

---

## 3. `/metrics` — KPI 대시보드 설계

```
/metrics [대상]
```

### 4단계 메트릭스 프레임워크

| 메트릭 유형 | 수량 | 역할 | 예시 |
|------------|------|------|------|
| **North Star** | 1개 | 성공의 단일 지표 | 월간 재구매율 |
| **인풋 메트릭** | 3~5개 | NSM을 구동하는 레버 | 신규 방문자, 전환율, AOV |
| **헬스 메트릭** | 3~5개 | 안정적이어야 하는 지표 | 반품률, CS 응답시간, 재고회전 |
| **카운터 메트릭** | 1~2개 | 잘못된 최적화 방지 | 할인율 (매출만 올리면 이익 감소) |

### 알림 임계치 (Green / Yellow / Red)

각 메트릭별로 3단계 기준을 설정하여 자동 모니터링.

---

## 4. 매출 분석 — `sales-analysis`

```
"매출 분석해줘"
"이번 주 실적 어때?"
"채널별 비교해줘"
```

### 분석 관점

| 관점 | 내용 |
|------|------|
| **채널별** | 자사몰·무신사·플래그십·면세·해외 비교 |
| **카테고리별** | UNI·WOMAN·용품 매출 믹스 |
| **SKU별** | 히트상품·부진상품 식별, 소진율 |
| **기간별** | WoW, MoM, YoY 비교 |
| **BTA별** | Basic·Trend·Accent 매출 구성비 |

### 기존 대시보드와 연계

FPOF에 이미 구축된 대시보드:

| 대시보드 | 위치 |
|---------|------|
| 매출 대시보드 | `workspace/26SS/dashboard/board_sales.html` |
| 상품 상세 대시보드 | `workspace/26SS/dashboard/board_product-detail.html` |
| 마스터 리뷰 | `workspace/26SS/dashboard/board_master-review.html` |

---

## 5. `/cohorts` — 코호트 분석

```
/cohorts [분석 데이터]
```

### 분석 내용

| 분석 | 설명 |
|------|------|
| **리텐션 테이블** | 월별 가입 코호트의 재방문/재구매 추이 |
| **재구매 패턴** | 첫 구매 → 2차 구매 전환율, 평균 재구매 주기 |
| **LTV 추정** | 코호트별 고객 생애 가치 |
| **이탈 패턴** | 어느 시점에 가장 많이 이탈하는가 |
| **채널별 차이** | 자사몰 vs 무신사 코호트 품질 비교 |

---

## 6. `/ab-test` — A/B 테스트 분석

```
/ab-test [실험 데이터]
```

### 분석 내용

| 항목 | 설명 |
|------|------|
| **통계적 유의성** | p-value, 신뢰구간 계산 |
| **효과 크기** | 실제 비즈니스 임팩트 (단순 유의성이 아닌) |
| **의사결정 권고** | 채택/기각/추가 테스트 + 근거 |
| **세그먼트 분석** | 세그먼트별 효과 차이 (모바일 vs 데스크톱 등) |
