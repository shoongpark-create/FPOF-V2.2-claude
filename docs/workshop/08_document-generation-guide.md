# 세션 8: 문서 생성 3종 세트 — `/doc` `/sheet` `/pdf`

> **소요시간**: 30분 (설명 10분 + 데모 10분 + 실습 10분)
> **대상**: 전 멤버
> **목표**: 워드·엑셀·PDF 문서를 AI로 자동 생성하는 3가지 명령어를 실습

---

## 1. 문서 4총사 — 한눈에 비교

| 명령어 | 출력 | 용도 | 특징 |
|--------|------|------|------|
| `/doc` | `.docx` | 기획서·보도자료·브리프 | 목차 자동 생성, A4 레이아웃 |
| `/sheet` | `.xlsx` | 라인시트·OTB·매출 데이터 | 수식·필터·조건부 서식 자동 |
| `/pdf` | `.pdf` | 바이어용·인쇄용 보고서 | 브랜드 표지·헤더/푸터·페이지 번호 |
| `/deck` | `.pptx` | 프레젠테이션 | (세션 7 참조) |

---

## 2. `/doc` — 워드 문서 생성

### 기본 사용법

```
/doc [유형]
```

유형 생략 시 현재 PDCA 단계에 맞는 문서를 추천합니다.

### PDCA 단계별 문서 유형

| 단계 | 유형 | 내용 |
|------|------|------|
| **Plan** | `trend-brief` | 트렌드 브리프 — 시장 분석, 키워드, 방향성 |
| | `season-plan` | 시즌 기획서 — 테마, 컨셉, 전략, 일정 |
| | `brand-brief` | 브랜드 브리프 — 포지셔닝, 타겟, 핵심 메시지 |
| **Design** | `design-brief` | 디자인 브리프 — 컨셉, 무드, 레퍼런스 |
| | `design-review` | 디자인 리뷰 — 디자인별 평가, 수정사항 |
| **Do** | `campaign-plan` | 캠페인 기획서 — 전략, 채널, 콘텐츠, 일정 |
| | `press-release` | 보도자료 — 브랜드 스토리, 시즌 하이라이트 |
| | `pdp-copy` | 상품 상세 카피 — 상품명, 설명, 소재 정보 |
| | `production-spec` | 생산 사양서 — 소재, 봉제, QC 기준 |
| **Check** | `season-review` | 시즌 리뷰 — 성과 분석, 교훈, 제안 |
| | `playbook` | 플레이북 — 재현 가능한 성공/실패 패턴 |

### 자동 적용 스타일

- **표지**: 문서 제목, 시즌, 날짜, 작성자(에이전시명)
- **본문**: 제목(H1) → 섹션(H2) → 소제목(H3) 계층 구조
- **톤앤매너**: 내부 문서 = professional / 외부 문서 = brand voice
- **페이지**: A4, 여백 2.5cm, 목차 자동 생성

### 실습

```
/doc campaign-plan
```

## 3. `/sheet` — 엑셀 시트 생성

### 기본 사용법

```
/sheet [유형]
```

### PDCA 단계별 시트 유형

| 단계 | 유형 | 내용 |
|------|------|------|
| **Plan** | `line-sheet` | 라인시트 — SKU 목록, 사이즈/컬러 비율, 가격 |
| | `otb` | OTB — 카테고리별 예산 배분, 수량 계획 |
| | `category-mix` | 카테고리 믹스 — SKU 수, 매출 비중 계획 |
| **Design** | `cost-estimate` | 원가 견적서 — 원부자재, 가공비, 원가율 |
| | `color-size` | 컬러/사이즈 — 색상-사이즈 매트릭스 |
| **Do** | `bom` | BOM — 원부자재 명세, 단가, 소요량 |
| | `production` | 생산 오더시트 — 오더 수량, 납기, 선적 |
| | `marketing-budget` | 마케팅 예산 — 채널별 배분, ROAS 목표 |
| **Check** | `sales-data` | 매출 데이터 — 채널/카테고리/SKU별 매출 |
| | `kpi-tracker` | KPI 추적 — 목표 vs 실적, 달성률 |

### 자동 적용 기능

| 기능 | 설명 |
|------|------|
| **시트 탭 구조** | 요약 → 상세 데이터 → 차트 데이터 |
| **필터** | 컬럼 헤더에 자동 적용 |
| **수식** | 합계/소계 자동 삽입 |
| **조건부 서식** | 달성률 색상 코딩 (Green/Yellow/Red) |
| **숫자 포맷** | 통화 ₩, 비율 %, 천단위 구분 |
| **인쇄 설정** | 인쇄 영역, 반복 헤더 |

### 실습

```
/sheet line-sheet
```

## 4. `/pdf` — PDF 보고서 생성

### 기본 사용법

```
/pdf [유형]
```

### PDF 유형

| 카테고리 | 유형 | 내용 |
|---------|------|------|
| **시즌 문서** | `trend-report` | 트렌드 분석 보고서 |
| | `season-book` | 시즌 북 — 컨셉·디자인·상품 통합 |
| | `line-sheet` | 바이어용 라인시트 (이미지·스펙·가격) |
| **생산 문서** | `techpack` | 테크팩 — 도식화, BOM, 사이즈 스펙 |
| | `cost-sheet` | 원가 산출서 |
| **마케팅** | `campaign-brief` | 캠페인 브리프 |
| | `press-kit` | 프레스 키트 |
| **분석** | `season-report` | 시즌 종합 보고서 |
| | `kpi-dashboard` | KPI 대시보드 보고서 |

### 자동 적용 요소

- **표지**: 브랜드 로고 영역, 시즌명, 문서 제목, 날짜
- **목차**: 자동 생성
- **헤더/푸터**: 브랜드명, 페이지 번호, 보안 등급
- **브랜드 컬러**: visual-identity.json 기반

### 실습

```
/pdf season-report
```

## 5. 연계 워크플로우

### 분석 → 문서 → 프레젠테이션 파이프라인

```
[Step 1] 매출 데이터 분석
[Step 2] /sheet sales-data → 엑셀로 데이터 정리
[Step 3] /doc season-review → 워드로 시즌 리뷰 문서
[Step 4] /pdf season-report → PDF로 인쇄용 보고서
[Step 5] /deck report → PPTX로 경영진 발표
```

### 바이어 미팅 준비

```
/sheet line-sheet → 바이어용 라인시트 엑셀
/pdf line-sheet → 바이어용 라인시트 PDF (이미지 포함)
/deck buyer → 바이어 프레젠테이션
```
