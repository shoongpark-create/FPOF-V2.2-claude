# 세션 10: 경쟁 분석 & 전략 프레임워크

> **소요시간**: 35분 (설명 15분 + 데모 10분 + 실습 10분)
> **대상**: 전략기획·MD·브랜드 매니저
> **목표**: 4대 전략 프레임워크와 경쟁 분석 도구를 실습

---

## 1. 전략 분석 스킬 전체 구조

```
/market-scan ── SWOT + PESTLE + Porter's + Ansoff (4개 통합)
                  ↓
/competitive ── 경쟁사 심층 분석 (직접/간접 경쟁사)
                  ↓
/battlecard ── 세일즈용 경쟁 비교표
```

---

## 2. `/market-scan` — 거시환경 종합 분석

### 사용법

```
/market-scan [대상]
```

대상 생략 시 와키윌리 현재 시즌 기준.

### 4대 프레임워크 한 번에 실행

| 프레임워크 | 분석 내용 |
|-----------|----------|
| **SWOT** | 강점·약점·기회·위협 + 크로스 전략 (S×O, W×T) |
| **PESTLE** | 정치·경제·사회·기술·법률·환경 요인 + 영향도·시간축 |
| **Porter's 5 Forces** | 경쟁 강도·공급자·구매자·대체재·신규 진입 → 산업 매력도 |
| **Ansoff Matrix** | 시장 침투·시장 개발·제품 개발·다각화 → 성장 기회 |

### 크로스 프레임워크 종합 (핵심 차별점)

4개 프레임워크를 개별 실행하는 것과 다른 점:

- **수렴 시그널**: 여러 프레임워크가 공통 지적하는 포인트
- **전략적 필수 과제**: 분석 전반에서 긴급한 액션
- **핵심 리스크**: 완화해야 할 위협
- **최적 성장 기회**: 리스크 대비 최고의 성장 플레이

### 실습

```
/market-scan 와키윌리 26SS 스트릿웨어 시장
```

---

## 3. `/competitive` — 경쟁사 심층 분석

### 사용법

```
/competitive [경쟁사명 또는 시장]
```

### 분석 구조

| 단계 | 내용 |
|------|------|
| **경쟁사 식별** | 직접 경쟁사 5개 + 간접 경쟁사 2~3개 |
| **개별 분석** | 포지셔닝, 타겟, 강점/약점, 가격, 최근 움직임 |
| **비교 매트릭스** | 기능/상품 비교표 |
| **포지셔닝 맵** | 2x2 매트릭스 시각화 |
| **차별화 기회** | 방어 가능하고 가치 있는 영역 |
| **전략 권고** | 강화 / 따라잡기 / 무시할 영역 |

### 실습

```
/competitive
```

(와키윌리 경쟁 브랜드 기준 자동 분석)

---

## 4. `/battlecard` — 경쟁 배틀카드

바이어 미팅·세일즈 현장에서 바로 쓸 수 있는 비교표.

```
/battlecard [경쟁사명]
```

### 배틀카드 구성

| 섹션 | 내용 |
|------|------|
| **경쟁사 요약** | 한 줄 설명, 강점, 약점 |
| **기능 비교표** | 항목별 우리 vs 경쟁사 |
| **가격 비교** | 카테고리별 가격대 |
| **반론 스크립트** | "경쟁사가 더 좋지 않나요?"에 대한 대응 |
| **윈/로스 패턴** | 어떤 상황에서 이기고 지는지 |

## 5. 개별 프레임워크 (필요시 단독 실행)

| 명령어 | 프레임워크 | 사용 시점 |
|--------|-----------|----------|
| `"PESTLE 분석해줘"` | 거시환경 6요인 | 해외 시장 진출 검토 시 |
| `"Porter's 5 Forces 분석해줘"` | 산업 경쟁 구조 | 신규 카테고리 진입 시 |
| `"Ansoff 매트릭스 해줘"` | 성장 전략 | 사업 확장 방향 결정 시 |
| `/business-model` | BMC 9블록 | 비즈니스 모델 정리 시 |
| `/strategy-canvas` | 9섹션 전략 캔버스 | 전략 한 장 정리 시 |
| `/value-prop` | JTBD 밸류 프로포지션 | "왜 우리 옷을 사야 해?" |
