# 세션 1: 비케이브 리워크 프로세스 혁신 가이드

> **소요시간**: 30분 (설명 20분 + 토론 10분)
> **대상**: 전 멤버
> **목표**: 전사 방향성 5대 전환을 이해하고, 각자 업무에 적용할 체크리스트를 확인

---

## 1. 배경

2026년 3월 경영진 회의 3건(전사통합 W11 경영진 정렬회의, 비케이브 경영회의, 전체 관리자 회의)의 핵심 결정사항을 종합한 **전사 방향성 가이드**입니다.

### 한 문장 요약

> **"AI 시대, 뾰족한 브랜드와 빠른 의사결정으로 생존하고 성장한다."**

---

## 2. 5대 전환 — 핵심 변화 방향

| # | FROM (기존) | TO (전환) | 핵심 메시지 |
|---|-----------|---------|-----------|
| 1 | 기능 조직 (기획·디자인·영업·물류) | **브랜드 중심 조직** | 브랜드 책임자가 PMD 방향성 통합 관리 |
| 2 | AI = 효율 개선 도구 | **AI = 조직을 다시 짜는 것** | 선택이 아닌 **생존** |
| 3 | 안전하고 무난한 상품·콘텐츠 | **뾰족하고 감도 높은 브랜드** | SPA와 차별화 없이는 생존 불가 |
| 4 | 면세·할인 의존 유통 | **해외 직진출 + 내수 손익 관리** | 면세 점진적 축소, B2C 전환 |
| 5 | 분절적 상품 기획 | **히트상품 원팀 발굴 프로세스** | 영업·마케팅·브랜드가 원팀으로 리소스 집중 |

## 3. 브랜드 감도 강화

### 왜 감도인가?
- SPA(무신사 스탠다드, 유니클로)의 영역 확장 → **차별화 없이는 생존 불가**
- 상품 70점을 마케팅이 90점으로 올려야 하나, 현재 두 가지 모두 미흡
- 특정 브랜드가 아닌 **전사적 공통 과제**

### 영역별 전환 방향

| 영역 | 현재 문제 | 전환 방향 |
|------|----------|----------|
| **상품** | 너무 안전하고 무난함 | 일부 카테고리 **강제로 뾰족한 상품 편성**, 실패 감수 |
| **마케팅** | 안정 지향적 콘텐츠 | 논란 가능성 있어도 **트렌드 따라 밀어붙이기** |
| **스태프 스타일링** | 유니폼처럼 획일적 | 라인별 **다양화** |
| **콘텐츠** | 셀럽 의존, 자체 서사 부족 | **크루·커뮤니티·로컬 기반 세계관** 구축 |
| **속도** | 헤비한 콘텐츠에 시간 소모 | 빠르게 쪼개서 **스피디하게 실행** |

### 와키윌리 핵심 피드백
- 아이돌이 입을 만한 **트렌디한 상품 확대**
- **과감한 소재·디자인 시도**
- 브랜드 자체 서사·세계관·커뮤니티 콘텐츠 확보

## 4. AI 기반 조직 변혁

### 대표 핵심 메시지

> "AI 시대는 일 많이 하는 사람의 시대가 아니라, **판을 설계하고 선택하고 책임지는 사람**의 시대."

### 사람 vs AI 영역 구분

**사람만이 해야 할 일 (대체 불가)**:
- **전략적 의사결정** — 무엇을 할지/말지, 어떤 브랜드를 키울지
- **리스크 있는 판단** — 책임 있는 의사결정 (늦은 결정 = 한 달 기회 손실)
- **사람을 움직이는 일** — 조직 관리, 동기부여, 협업 촉진
- **창의력·감각·직관** — 트렌드 감지, 브랜드 감도, 시스템 설계

**AI로 대체·압축 가능한 영역**:
| 영역 | 구체적 업무 |
|------|-----------|
| 디자인실 | 레퍼런스 수집, 컬러 변형, 그래픽 작업 |
| 물류 | 운영 최적화 |
| 경영기획/전략실 | 레포트 생성, 데이터 분석 |
| 마케팅 | 단순 SNS·카피 생성, 트렌드 분석 자동화 |
| 기획·소싱 | 시즌 반복·레퍼런스 수집 자동화 |

### AI 시대에 귀해질 인재
1. 브랜드 정체성을 만들 수 있는 **디자이너**
2. 트렌드를 읽고 **크리에이터와 협업**할 수 있는 사람
3. PMD 방향성을 이해하고 관리할 수 있는 **키맨**
4. AI 활용 기반으로 업무를 **설계하고 실행**시킬 수 있는 사람
5. 유통 채널에 대한 **깊은 이해**를 가진 사람

## 5. AX(AI Transformation) 체계

### AI 도입 5단계

| 단계 | 내용 | 현재 상태 |
|------|------|----------|
| 1단계 | 생성형 AI 기본 활용 (보고서, 회의록) | ✅ 완료 |
| 2단계 | 보안/감사 통제 하 체계적 제공 | 🔄 진행 중 |
| 3단계 | 파일럿 검증 → 성과 평가 | 📋 계획 |
| 4단계 | 거버넌스 심사 후 전사 확산 | 📋 계획 |
| 5단계 | 인프라 투자 + 조직 확대 | 📋 계획 |

### AX 운영 원칙
- 현업이 만들고, IT가 보완
- **ERP/WMS 등 원천 데이터 변경·훼손은 AI 금지**
- AI/로코딩은 데이터 분석·업무 개선 용도로 제한
- 모든 결과물은 정기 심사 대상
- AI 생성 데이터는 **IT 인프라 종속 DB에 집중** → 전사 공유

### 거버넌스 프로세스

```
현업 TF 8~10명 (바이브코딩 → 프로토타입/POC)
    ↓
AX 거버넌스 커미티 심사 (경영진 참여, 격주/월)
    ↓
┌─ 확산: RPA → 서버 배포 / 현황판 → 사내 플랫폼(EIS)
├─ 홀딩: 요건 재정비 → 재심사
└─ 중단: 실패 보고서 작성 → 레퍼런스 축적
```

## 6. 히트상품 발굴 프로세스 혁신

### 기존 vs. 신규 프로세스

```
[기존] 분절적 프로세스
시즌 기획 금액 → 메인/QR 구분 → 상품 배치 → 마케팅 (순차적·분절적)

[전환] 원팀 프로세스
유효 스타일 발굴 → 의사결정 근거 기록
    ↓
VM·마케팅·영업 연계 → 리소스 집중 투입 (원팀)
    ↓
인플루언서 시딩 → 빠른 실행
    ↓
모니터링 & 의사결정 (실무단 + 책임자급 즉시 결정)
```

### 무신사 전략 재정립
- **부진의 본질**: 할인 프로모션 부족이 아닌, **크리에이터×브랜드×상품 삼각 협업 부재**
- 인플루언서가 **제작 시점부터 관여** → 영상 제작 → 채널 광고

## 10. 업무 시 자문 체크리스트

업무 의사결정 전 아래 5가지를 자문하세요:

- [ ] 이 결정이 **브랜드 감도**를 높이는가?
- [ ] **AI로 대체 가능한 업무**에 사람 리소스를 쓰고 있지 않은가?
- [ ] **데이터 근거**가 있는 판단인가?
- [ ] **속도**가 충분한가? (늦은 결정 = 한 달의 기회 손실)
- [ ] **손익**을 함께 고려했는가? (매출만이 아닌 이익·재고까지)

### 직급별 핵심 행동 기준

| 직급 | 핵심 행동 |
|------|----------|
| **경영진** | 빠른 의사결정, 데이터 해석, 시스템 설계, 인재밀도 관리 |
| **관리자** | 프로세스 최신화, 조직관리, 감도 판단, 유관부서 소통, 상업적 감각 |
| **팀원** | 감도, 실행력, 속도, AI 활용 능력 |
