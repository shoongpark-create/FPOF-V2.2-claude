OKR & 스프린트 관리

/okrs → /roadmap → /stakeholders → /sprint plan · retro · release

30분 관리자 · 리더

🎯 분기 사이클

/okrs → /roadmap → /stakeholders → /sprint plan → /sprint retro

/okrs — 5대 경영목표 연결

  • 목표(O) = 정성적, 영감적
  • 핵심결과(KR) = 정량적, 측정 가능
  • 70% 달성 = 잘 보정된 목표
  • 게이밍 방지 카운터 메트릭

/sprint — 3가지 모드

  • plan — 용량 배분 + 스토리 선택
  • retro — Start/Stop/Continue, 4Ls, Sailboat
  • release — 릴리즈 노트 자동 생성

  • 고객 — 해결 못 하는 문제
  • 경쟁 — 안 하고 있는 것
  • 기술 — 새로 가능해진 것

  • Tigers — 확실히 위험 → 즉시 완화
  • Paper Tigers — 위험해 보이지만 아님
  • Elephants — 모두 알지만 말 안 하는 것
