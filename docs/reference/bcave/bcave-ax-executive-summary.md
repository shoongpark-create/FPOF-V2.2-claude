## 1. 배경: 왜 지금 AX인가

### 대표이사 방향 (요약)

> "업무 경쟁력 = 업(業) 이해 + AI 실행력"
> "AI를 사용하는 조직이 아니라, AI를 기반으로 일하는 조직이 되어야 합니다."

### 패션 산업 AI 전환 현황

- **Zara**: AI 수요 예측 + 재고 최적화로 폐기율 30% 감축, AI 촬영 도입
- **H&M**: AI 클론 모델 마케팅, 수요 예측으로 재고 최적화
- **SHEIN**: SNS 트렌드 → 3일 제작, 일 수천 신상품 출시
- **국내**: 삼성물산·LF·한섬 AI 전환 본격화, 스튜디오랩 '젠시' CES 최고혁신상

**결론:** 중견 패션사가 선제적 AX를 하지 않으면 경쟁력 확보의 기회를 놓침.

## 2. 비케이브의 AX 포지션

### 강점

| # | 강점 | 활용 방안 |
|---|------|----------|
| 1 | **CEO의 강력한 톱다운 의지** | AX 추진의 최대 동력, 조직 저항 극복 |
| 2 | **BTA라는 공통 언어** | P-M-D 부서 간 이미 표준 프레임워크 → AI가 학습할 기반 |
| 3 | **FPOF 프로토타입** | 와키윌리에서 검증 완료 → 전사 확장 템플릿 |
| 4 | **시즌 PDCA 사이클** | 반복 구조 → AI 자동화에 최적 |
| 5 | **강소기업 규모 (영업이익 220억)** | 의사결정 빠름, 실험 비용 낮음, 대기업보다 빨리 움직일 수 있음 |

### 과제

- 데이터 인프라 미흡 (CEO 인정) → "있는 데이터로 즉시 시작"이 핵심
- IT 전담 인력 부족 → 현장 챔피언 + FPOF 시스템으로 해소
- "패션은 감이야" 문화 → "감을 없애는 것이 아니라 근거를 더하는 것"으로 프레이밍

## 3. 핵심 전략: 행동 변화 3레이어

```
AX ≠ 도구 도입     AX = 행동 변화

Layer 3. 조직 시스템  →  데이터 기반 의사결정, 프로세스 자동화
Layer 2. 팀 워크플로우  →  P-M-D별 AI 에이전트, BTA 통합 AI 연동
Layer 1. 개인 역량     →  AI 리터러시, 도파민 경험, 챔피언 육성
```

### 핵심 메커니즘: 챔피언 + 도파민

- **집체 교육은 비효율** → 의지 높은 **챔피언 1-2명/부서**를 먼저 변화시킴
- 챔피언의 성과가 동료에게 **FOMO** 유발 → 자발적 확산
- **바텀업(챔피언)** + **톱다운(CEO 방향)** = 가장 효과적인 AX 실행

## 4. 실행 로드맵 (3단계)

### Phase 1: Quick Win (4-6월, 3개월)

> 목표: AI 도파민 경험, 챔피언 발굴, 즉각적 효율 개선

| 핵심 액션 | 기대 효과 |
|----------|----------|
| 부서별 챔피언 1-2명 모집 (10-15명) | 자발적 변화 리더 확보 |
| 전 직원 AI 리터러시 L1 교육 (2시간) | 기본 이해도 확보 |
| **주간 매출 리포트 자동화** | 주 2시간 → 15분 |
| **회의록 AI 자동 정리** | 즉시 액션 아이템 생성 |
| **경쟁사 SNS 자동 모니터링** | 매일 3사 자동 수집·요약 |
| Teams `#ax-문제은행` 개설 | 현장 문제 발굴 시작 |

**투자:** 최소 (기존 AI 도구 활용 + 내부 인력 시간 배분)
**리스크:** 낮음 (실패해도 학습 효과)

### Phase 2: 정착 (7-12월, 6개월)

> 목표: AI 에이전트 구축, 데이터 인프라 정비, 커버낫 FPOF 확장

| 핵심 액션 | 기대 효과 |
|----------|----------|
| 부서별 AI 에이전트 5-8개 구축 | 반복 업무 50-80% 자동화 |
| 데이터 표준화 (BTA 통합시트 디지털화) | AI 분석 기반 확보 |
| FPOF 커버낫 확장 | 대표 브랜드 AI 오케스트레이션 |
| 27SS 시즌 기획에 AI 트렌드 분석 첫 적용 | 데이터 기반 BTA 설계 |
| AI 기반 주간 리뷰 프로세스 전면 도입 | 실시간 의사결정 지원 |

### Phase 3: 확산 (2027년, 12개월)

> 목표: 전 브랜드 FPOF, AI 네이티브 의사결정, 비즈니스 임팩트

| 핵심 액션 | 기대 효과 |
|----------|----------|
| FPOF 리(Lee) + 기타 브랜드 확장 | 3개 이상 브랜드 AI 운영 |
| AI 수요 예측 v1 (베이직 아이템) | 재고 효율 15% 개선 |
| 전사 AI 네이티브 시즌 기획 확립 | 기획 리드타임 30% 단축 |
| AI 거버넌스 체계 확립 | 데이터 보안·사용 기준 정립 |

## 5. P-M-D 임팩트 요약

| 영역 | As-Is | To-Be | 효과 |
|------|-------|-------|------|
| **P (상품)** | BTA 비율 감(感) 기반 설정, CAD 수작업 체크 | AI 시뮬레이션 기반 BTA 설계, 자동 3B 감지 | 기획 시간 60% 단축, 의사결정 근거 투명화 |
| **M (마케팅)** | B.A.M.P 수작업 검증, SNS 균형 감각 의존 | AI 자동 검증, 3균형 모니터링, 카피 초안 자동 | 콘텐츠 생산 2배, 마케팅 품질 일관성 |
| **D (유통)** | 매장 라운딩 사진 → 구두 피드백 | AI 사진 분석 → 자동 VM 스코어 + 개선안 | 라운딩 효율 60% 향상, 매장 품질 표준화 |

## 6. 성공 지표 (Phase별)

| Phase | 핵심 KPI | 목표 |
|-------|---------|------|
| **Phase 1** | 챔피언 활성화, AI 활용 건수, Quick Win 완료율 | 부서당 1명+, 50건/분기, 80% |
| **Phase 2** | 리포트 시간 단축, AI 에이전트 수, 데이터 의사결정 비율 | 50%↓, 8개+, 40%+ |
| **Phase 3** | 재고 효율, 콘텐츠 속도, 기획 리드타임, 전사 AI 활용률 | 15%↑, 2배, 30%↓, 70%+ |

## 7. 투자 & 리소스 요청

| 항목 | Phase 1 | Phase 2 | Phase 3 |
|------|---------|---------|---------|
| **인력** | 전략기획 AX 리드 겸임 1명 | AX 리드 전담 전환 검토 | 유지 or 확대 |
| **예산** | 최소 (기존 도구 활용) | AI 솔루션 라이선스, 외부 교육비 | 데이터 인프라 투자 |
| **시간** | 챔피언: 주 2-3시간 | 챔피언: 주 3-5시간 | 일상 업무 통합 |
| **외부 지원** | AI 리터러시 강사 (1회) | 데이터 표준화 컨설팅 (선택) | 수요 예측 솔루션 (선택) |

**Phase 1은 추가 예산 거의 불필요** — CEO의 방향 설정 + 내부 의지가 핵심 투자.

## 9. 한 줄 요약

> **비케이브의 AX는 "패션을 잘 아는 사람 + AI"의 결합입니다.**
> 감(感)을 없애는 것이 아니라, 감에 데이터 근거를 더하고,
> 반복 업무는 AI가 처리하여 사람은 판단과 창의에 집중하는 조직을 만듭니다.
>
> Phase 1 투자는 최소이고, 리스크는 낮으며, CEO의 이미 표명된 방향과 완전히 일치합니다.
> **지금 시작하지 않을 이유가 없습니다.**
