# 와키윌리 브랜드 방향성 브레인스토밍

> [!info] 문서 목적
> Lee 27SS 디자인 방향성 피드백의 4축 프레임워크를 와키윌리에 맞게 전환하고, 대표이사의 전사 방향성(뾰족함·감도·AI·히트상품)을 26SS·26FW 마케팅과 27SS 상품 개발에 구체적으로 반영하기 위한 브레인스토밍 문서입니다.

> [!summary] 핵심 전제 — 대표이사 메시지 3줄 요약
> 1. **"뾰족함이 없는 브랜드는 무조건 죽는다"** — SPA와 차별 없는 무난함은 생존 불가
> 2. **"셀럽은 확성기일 뿐"** — 내부 서사·세계관이 먼저, 빅셀럽 의존 체질 탈피
> 3. **"상품 70점을 마케팅이 90점으로 올려야 한다"** — 상품과 마케팅 동시 감도 상향

## Part 1. 와키윌리 4축 프레임워크 (Lee 4축의 WW 전환)

> Lee의 UNI/WOMEN/DENIM/ARCHIVE 4축을 와키윌리 정체성에 맞게 재구성

### 축 A. TREND UNI (트렌드 유니섹스) — 매출 엔진

| 항목         | 방향                                     |
| ---------- | -------------------------------------- |
| **포지션**    | 한국 유행 기반, 성별 중립, 20대 초중반 타겟            |
| **핵심 키워드** | 아이돌이 입을 만한 / 릴스에 올릴 만한 / 무신사에서 눈에 띄는   |
| **상품 방향**  | 벨루아 셋업, 크롭 실루엣, 오버핏 변형, 메시·시스루, 컬러 블로킹 |
| **주의**     | 기본물에 몰입 금지. 시즌 히트 아이템 반드시 생성           |
| **마케팅 연계** | 인플루언서 시딩 + 스타일링 콘텐츠 선행 → 판매 연결         |

### 축 B. WOMEN (우먼스 확장) — 성장 드라이버

| 항목         | 방향                                            |
| ---------- | --------------------------------------------- |
| **포지션**    | 여성스러운 실루엣 강화, CK·게스 대비 차별화                    |
| **현황**     | 발주 25% → 매출 비중 50%, 판매율 2배 효율 (**검증된 성장 동력**) |
| **상품 방향**  | 러플라인 프로젝트, 리본 소재, 크롭 가디건, 페미닌 자켓, 반팔 니트       |
| **핵심 전략**  | 지젤 캠페인 착장 호조 → QR 즉시 대응 → FW 간절기로 연결          |
| **마케팅 연계** | 자사몰 우먼스 전용 상품 월별 기획, 여성 인플루언서 그루핑             |

### 축 C. KIKI (키키 IP·캐릭터 자산) — 브랜드 아이덴티티

> Lee에서 DENIM이 브랜드 핵심 자산이듯, 와키윌리에서는 **키키(KIKI)**가 핵심 자산

| 항목         | 방향                                              |
| ---------- | ----------------------------------------------- |
| **포지션**    | 키키를 중심으로 와키윌리만의 세계관 구축                          |
| **상품 방향**  | 키키 그래픽 과감한 활용(스프레이, 엠보스, 톤톤), IP 콜라보, 시즌별 키키 변형 |
| **로고 운영**  | 심볼(키키) 형태 고정 / 컬러·크기·위치·기법 자유 변형 가능             |
| **마케팅 연계** | 키키 크루·커뮤니티, 키키 스토리텔링 콘텐츠, 성수 플래그십 키키 체험 공간      |
| **주의**     | 상품용은 엄격 적용(영구적), 마케팅용은 유연(밈·트렌드 반영 가능)          |

### 축 D. CULTURE (크루·커뮤니티·세계관) — 브랜드 자생력

> Lee의 ARCHIVE(헤리티지)에 대응하는 와키윌리의 축: **문화 자산 축적**

| 항목        | 방향                                    |
| --------- | ------------------------------------- |
| **포지션**   | 셀럽 없이도 브랜드 자체 서사·세계관으로 팬덤 구축          |
| **운영 방향** | 와키윌리 크루 운영, 로컬 커뮤니티 이벤트, 성수 플래그십 프로그램 |
| **콘텐츠**   | 브랜드 자체 세계관 영상, 키키 유니버스, 스태프 스타일링 다양화  |
| **타겟**    | 매니아·코어 팬층 별도 관리                       |
| **핵심**    | "빅셀럽 확성기" 이전에 "뭘 말할 것인가"를 먼저 갖추기      |

## Part 2. 26SS 마케팅 — 즉시 반영 (4~6월)

> 지금 즉시 실행 가능한 것 중심. 상품은 이미 생산·입고 완료이므로 **마케팅·VM·채널 운영**에 집중

### 2-1. 긴급 과제: 기본물 재고 소진 (65억/7.5만 장)

| 전략 | 구체안 | 우선순위 |
|------|--------|---------|
| **글로벌 소진** | 대만 반팔 중심 10~15억 공급, 중국 GB 테스트 즉시 세팅 | 🔴 |
| **내수 채널 분리** | 아울렛·행사장·온라인 계획전에 한정 → 정상 매장 가격 보호 | 🔴 |
| **스타일링 리프레시** | 기본물을 트렌드 아이템과 크로스 스타일링 → 콘텐츠용으로 재가치화 | 🟠 |
| **1물 1가 원칙** | 일괄 30% 할인 반대. 매출원가 훼손 방지 | 🔴 (대표 방침) |

### 2-2. 우먼스 QR 가속

| 액션 | 내용 |
|------|------|
| 지젤 화보 착장 8개 SKU 리오더 완료 | 판매 모멘텀 즉시 이어가기 |
| 반팔 T 최우선 QR 대응 | 소재 사전 부킹 완료 → 4~5주 입고 |
| 반팔 니트 현물원사 기반 빠른 대응 | 간절기 전환 시점 활용 |
| 자사몰 우먼스 전용 월별 상품 기획 | 온라인 매출 회복 핵심 |

### 2-3. 마케팅 콘텐츠 방향 전환

| FROM (현재) | TO (전환) |
|------------|----------|
| 안전하고 무난한 콘텐츠 | **트렌드 밀어붙이기, 반응 있는 콘텐츠 우선** |
| 빅셀럽 의존 | **브랜드 자체 크루·커뮤니티 콘텐츠 + 마이크로 인플루언서** |
| 헤비한 콘텐츠에 시간 소모 | **빠르게 쪼개서 스피디하게 실행** |
| 셀럽 제외 시 콘텐츠 공백 | **키키 IP 활용, 성수 플래그십 체험, 스태프 스타일링** |

### 2-4. 성수 플래그십 활용 (26SS 집중)

| 시기 | 액션 |
|------|------|
| **4월** | 3월 오픈 프로모션 리뷰 → 반응 부진 항목 즉시 피버팅 |
| **5월** | 체험공간 전환 — 앞마당 활용, 참여형 이벤트, SNS 자발적 공유 유도 |
| **상시** | 풍선 GWP 폐지 → **쓸 수 있는 GWP** 또는 **즉각적 혜택** 전환 |

### 2-5. 무신사 전략 재정립

| 현재 문제 | 전환 방향 |
|----------|----------|
| 할인 프로모션 부족이 부진 원인? | **NO** — 크리에이터×브랜드×상품 삼각 협업 부재가 본질 |
| 기획전 상품 배열 영업 주도 | 기획 의도 반영 가능한 **링크 역할(온라인 바이어)** 필요 |
| 무신사 전용 SMU | **반대** (대표 방침) → 자사몰/플래그십 전용 챌린지 상품은 가능 |
| 인플루언서 시딩 | 제작 시점부터 인플루언서 관여 → 영상 제작 → 채널 광고 |

### 2-6. IMC 실행 원칙

> 대표 피드백: "반응 없으면 즉시 피버팅. 대표 지시를 경직하게 해석하지 말 것"

- IMC 주력 스타일 현황 **1시트** 운영 — 실시간 모니터링
- 리오더 스타일 체크업 **2시트** 운영
- 마케팅 투자 대비 효율 측정 **대시보드** 구축
- **인플루언서 그루핑**: 크루 / 셀럽 / 마이크로 → 스타일별 매칭

### 3-2. 축별 FW 마케팅 키 액션

#### A. TREND UNI — FW 히트 아이템 발굴

- 간절기(9~10월): 크롭 바람막이, 숏 패딩, 오버핏 아우터
- 성수기(11~12월): 패딩·플리스·무스탕 등에서 **와키윌리다운 한 방** 아이템
- 인플루언서 제작 시점 관여 → 시딩 → 영상 → 광고 파이프라인 구축
- 매장에 "한 눈에 꽂히는" 아이템 반드시 편성

#### B. WOMEN — 우먼스 본격 볼륨업

- SS 러플라인·페미닌 라인 호조 → FW 가디건·크롭 아우터·니트 드레스로 확장
- 상품→마케팅→매장 구현까지 **소비자 여정 전체 설계**
- 백화점 여성 비중 31% → **40% 목표**로 FW 확대 설계
- 여성 인플루언서/크리에이터 전용 협업 라인

#### C. KIKI — 키키 캠페인 본격화

- FW 시즌 키키 그래픽 컬렉션 드롭 (한정판/소량)
- 키키 × 외부 아티스트/크리에이터 콜라보
- 성수 플래그십 키키 팝업 공간 → SNS 자발적 공유 극대화
- 키키 스티커/패치/악세서리 등 소품으로 진입 장벽 낮추기

#### D. CULTURE — 브랜드 자생력 기반 구축

- 와키윌리 크루 1기 모집 및 운영 (20대 초중반, 10~20명)
- 크루 기반 스트릿 스냅, 로컬 이벤트, 커뮤니티 콘텐츠
- 성수 플래그십을 문화 허브로 활용 (음악·아트·패션 교차)
- 셀럽 없이도 돌아가는 콘텐츠 루프 만들기

## Part 4. 27SS 상품 개발·디자인 반영 포인트

### 4-1. Lee 피드백에서 가져올 프로세스 (와키윌리 전환)

Lee에서 제안된 상품화 프로세스를 와키윌리에 맞게 재구성:

```
세련된 스타일링 내부 합의
    ↓
아이템 발라내기 (트렌드 풀에서 와키윌리향 선별)
    ↓
라인/스타일 픽스 (4축 중 어디에 해당하는지 태깅)
    ↓
트렌드 부합 검토 (WGSN + 무신사/인스타 데이터)
    ↓
일부 아이템에 키키 IP·와키윌리 디테일 적용
    ↓
트렌드와 브랜드 아이덴티티의 결합 확인
    ↓
히어로 아이템 선정 (카테고리당 2~3개)
```

### 4-2. 27SS 디자인 가이드라인

#### 컬러 전략

| 레이어 | 내용 | 근거 |
|--------|------|------|
| **코어 컬러** | 와키윌리 DNA 컬러 재정립 필요 (키키 옐로우, 블랙, 화이트 등) | Lee 피드백 "코어~트렌드 체계 부재" 동일 이슈 |
| **시즌 컬러** | WGSN + 경쟁사 + 판매 데이터 근거로 제시 | 감각이 아닌 데이터 기반 |
| **Accent 컬러** | 트렌드 관점에서 과감하게 (네온, 파스텔, 컬러 블로킹) | "예쁨의 문제가 아니라 체계 부재" |

#### 뉴베이직 27SS 설계 (5대 차별화 요소)

> 대표 방침: "컬러·배색·디테일로만 규정하면 그것만 나온다"

| 차별화 요소 | 와키윌리 적용 방향 | 레퍼런스 |
|------------|------------------|---------|
| 1. **원단** | 와키윌리만의 터치감 — 피치워싱, 헤비 코튼, 테크 소재 믹스 | 스튜시 기본 후디 |
| 2. **컬러** | 코어 컬러 위에 시즌 악센트 컬러 얹기 | Lee 코어~트렌드 체계 |
| 3. **실루엣** | 크롭·세미오버·박시 등 시즌별 핏 방향 명확화 | 매 시즌 핏 가이드 필수 |
| 4. **로고 표현** | 키키 톤톤·엠보스·스프레이·오버사이즈 등 기법 다양화 | 나이키 스우시 운영법 참고 |
| 5. **디테일** | 스티칭·라벨·지퍼풀·패치 등에서 와키윌리 정체성 | 워크웨어 영감 활용 가능 |

	- 예: 뉴베이직 맨투맨 2핏 + 후디 1핏 + 반팔 T 2품 + 조거 1핏 + 크롭 탑 1품
	- 각 아이템에 원단/부자재/봉제/핏/가격대/마진 목표/리오더 기준까지 문서화

## Part 5. 조직·프로세스 전제 조건

> 4축 프레임워크가 작동하려면 아래 조직·프로세스 전제가 필요

### 5-1. 카테고리 유닛과 4축의 연결

```
카테고리 유닛 4개                         4축 프레임워크
──────────────                         ──────────────
유니 아우터(우븐팬츠 포함)  ──→  주로 TREND UNI + 일부 KIKI
유니 이너바텀(다이마류)     ──→  주로 TREND UNI + 뉴베이직
우먼스                     ──→  주로 WOMEN + 일부 KIKI
액세서리(통합)             ──→  주로 KIKI + CULTURE

오퍼레이터 조직            ──→  4축 통합 관점에서 브랜드 일관성 관리
```

### 5-2. 주간 운영과 4축 점검

| 요일 | 기존 운영 | 4축 점검 추가 |
|------|----------|-------------|
| 월요일 | 개인 실행 결과 분석 → 팀별 미팅 | 각 유닛의 4축 반영도 자가 점검 |
| 화요일 | 카테고리 미팅 → 협업부서 조율 | 유닛 간 4축 크로스 체크 |
| 수요일 | 최종 의사결정 → 실행 집중 | 오퍼레이터 4축 통합 검토 |

### 5-3. 감도 인력 배치

> 대표 방침: "각 조직에 트렌드 민감형 인재 반드시 1명 배치"

- 각 카테고리 유닛에 **감도 담당 1명** 지정
- 릴스·쇼츠·무신사·인스타 트렌드 상시 모니터링
- 관리자는 아이디어 판단·실행 지원 역할
- 감도 약한 관리자에게는 감도 높은 짝꿍 배치로 보완

| 업무 | AI로 대체/압축 | 사람이 해야 할 일 |
|------|--------------|-----------------|

| 무신사/이국 리뷰 분석 | ✅ | 소비자 인사이트 도출 |
| 컬러·그래픽 변형 | ✅ | 최종 감도 판단 |
| 경쟁사 벤치마킹 | ✅ | 차별화 전략 수립 |
| 매출 시뮬레이션 | ✅ | 의사결정·리스크 판단 |
