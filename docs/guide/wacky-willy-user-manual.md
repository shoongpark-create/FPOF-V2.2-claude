## [1장] 시작하기

### 1.1 시스템이란 무엇인가?

#### FPOF = Fashion PDCA Orchestration Framework

**간단히 말해서:**
> 당신이 "무드보드 만들어줘"라고 말하면,
> AI가 알아서 크리에이티브 디렉터를 불러와서
> 무드보드를 만들어 주는 시스템입니다.

---

### 1.1 시스템이란 무엇인가?

#### 3가지 핵심 기능

**1️⃣ 자연어 커뮤니케이션**
- 복잡한 명령어 필요 없음
- 평소 말하듯이 말하면 됨
- AI가 알아서 이해하고 실행

**2️⃣ 자동 전문가 배정**
- 20명의 AI 전문가가 대기 중
- 말하는 내용에 따라 자동으로 적합한 전문가 배정
- 팀원을 직접 찾을 필요 없음

**3️⃣ 브랜드 보이스 보장**
- 와키윌리의 DNA, 톤앤매너가 사전에 세팅
- 모든 결과물이 브랜드에 완벽하게 맞음
- 일관된 품질 보장

## [2장] 기본 사용법

---

### 2.1 자연어로 말하는 법

#### 기본 원칙

**✅ 잘하는 방법**
```
"그래픽 티 무드보드 만들어줘"
"오버핏 후디 디자인 스펙 작성해줘"
"이번 시즌 트렌드 분석해줘"
```

**❌ 피해야 할 방법**
```
"알아서 해" → 너무 모호함
"그거 만들어" → 어떤 "그거"인지 모름
"뭐 할까?" → 스스로 결정하지 않음
```

---

### 2.1 자연어로 말하는 법

#### 구체적으로 말할수록 좋습니다

**❌ 너무 간단**
```
"디자인 해줘"
```
→ AI가 무엇을 디자인해야 할지 모름

**✅ 구체적**
```
"키키 캐릭터를 활용한 유니섹스 그래픽 티셔츠 디자인해줘.
타겟은 18~22세, 스트리트 감성으로"
```
→ 정확한 디자인 생성 가능

---

### 2.1 자연어로 말하는 법

#### 특정 전문가 지명하기

특정 전문가를 직접 부를 수 있습니다:

```
"패션 에디터한테 PDP 카피 맡길게"
"마케팅 디렉터가 GTM 계획 세워줘"
"시장 리서처한테 경쟁사 분석 시켜줘"
```

---

### 2.1 자연어로 말하는 법

## 용어 사전

### FPOF
**Fashion PDCA Orchestration Framework**
- 패션 브랜드 운영의 PDCA 사이클을 AI 에이전트 협업으로 자동화하는 프레임워크

### PDCA
**Plan → Design → Do → Check → Act**
- Plan: 기획
- Design: 디자인
- Do: 실행
- Check: 분석
- Act: 개선

### Quality Gate (QG)
- 각 PDCA 단계 전환 시 자동 실행되는 품질 검증 프로세스

### Brand Preset
